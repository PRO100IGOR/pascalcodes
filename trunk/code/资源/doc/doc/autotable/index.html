tab选项卡

  1. 引入resource/base/js/tool/tables.js

    这是一个jquery插件

    2. 假如页面上有一个table： 调用如下方法 
    $("#eptable").table({
		tid: //行的模版
		onAdd:function(tr){ //增加时,tr是增加的jquery对象
			
		},
		onDel:function(tr){  //删除时
			
		},
		noClick:true|false //用来屏蔽按钮的事件，如果为true，那么add_eptable等按钮无效
});
之后可以用$.tables["eptable"]获取到这个table对象.
如果页面上有id为add_eptable的按钮，那么点击这个按钮会调用table的增加一行方法。
如果页面上有id为del_eptable的按钮，那么点击这个按钮会调用table的删除一行方法。
如果页面上有id为up_eptable的按钮，那么点击这个按钮会调用table的上移一行方法。
