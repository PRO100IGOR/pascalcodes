jar包精简

  1. jar包共享于tomcat

    2. 1:新建的项目中，jar包只有几个必须的，其他的都不在项目中了。采用共享的方式放在tomcat/lib目录中.

    2:项目在eclipse中时，要引用系统级的library才能保证不报错。

    3：libaray和tomcat/lib所需的jar是一样的，下载：jar

    4:新建的项目中只有config.properties、accessories.properties两个配置文件了。

    5:在部署到tomcat中时，userlibrary中的jar也会复制到项目的lib中，但此时需要将lib中jar全部删掉，再将项目本身的lib中jar复制到tomcat项目的lib中

    6：当然，也可以完全不管上面所说的东西，只要在打包发布的时候分开就行了。但是在开发中就使用这样的方式，有助于提前发现问题。

    发布时应用系统中所需要的jar：lib
