盒子参数如下


	

		参数名参数说明
	

	

		panelWidth 盒子宽度，可以是数值或百分比。默认100%
	

	

		panelHeight  盒子高度，只能是数值。默认自适应内容
	

	

		whiteBg是否适应白色背景，该参数默认为false来适应右侧内页的背景色。这样当背景色不是白色也不会显得突兀。但如果是弹出窗口或其他情况，页面
		的背景色是白色，与右侧内页背景色不一致，则盒子周围会有一圈右侧内页的背景色，会显得突兀，这时可以将该参数设为true，以适应白色的背景。
	



弹窗中使用box1时要设置whiteBg="true"
