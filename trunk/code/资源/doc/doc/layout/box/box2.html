	还可通过参数实现不同的配置。参数如下：

	

	

		参数名参数说明
	

	

		panelWidth 盒子宽度，可以是数值或百分比。默认100%
	

	

		panelHeight  盒子高度，只能是数值。默认自适应内容
	

	

		panelTitle 盒子标题 默认为空
	

	

		overflow 设置是否出现滚动条。当设置高度时该参数生效。默认为hidden即不出现滚动条。当为auto时出现滚动条
	

	

		roller 只对某些皮肤有效。是否出现底部卷边效果，默认为true
	
	
	

		showStatus 是否显示右上方的文字标识，默认显示为”收缩“，这样默认可以收缩。如果为false则不显示标识，即不能收缩
	

	

		statusText自定义右上角标识。例如可以设置为“更多>>”
	
	
	

		panelUrl当自定义右上角标识后有效。用来设置点击右上角标识的链接地址。默认为"#"。此处也可调用JS的方法。
	

	

		panelTarget当自定义右上角标识后有效。用来设置点击右上角标识后新页面打开方式。默认值"_self"在本页打开。当设为"_blank"则在新页面打开
