窗口的皮肤的制可参照skins文件夹中的皮肤文件，首先给皮肤起个名称，然后创建一个相关的CSS文件，并建立一个和CSS文件名相同的文件夹放图片即可。lhgdialog皮肤结构相对非常简单，有一定的HTML和CSS设计基础即可完成设计工作。
