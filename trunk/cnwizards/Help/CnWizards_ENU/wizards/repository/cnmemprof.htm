CnMemProf Project Wizard

This wizard can generate a project with CnMemProf support. This project need CnMemProf.pas to compile and run, which is in CnWizards' Source directory. The dialog of this wizard is as below:

  • Popup Memory Usage Report Dialog when Exit: If this was selected, a message box will be show to report memory usage when program exit.

  • Log Objects: Whether log objects in CnMemProf.

  • Log Object Types using RTTI: If this was selected, CnMemProf will log more details of object using RTTI.

  • Log Memory Usage to File: Whether log memory usage to log file.

  • Log File: You can select memory log file name here. Empty means Memory.log in application directory.

Only Delphi is supported.
