MSDN Help Wizard

This wizard can use installed MSDN help to search word under cursor in MSDN from IDE.

Delphi/C++Builder are both supported. The default shortcut is Alt+F1.

If MSDN was not installed or version is unknown. Error messages will appear when executing this wizard.

MSDN menu is as below:

  • MSDN Help: Open MSDN and search the word under cursor in source editor. If no word under cursor, Index page will be opened.

  • MSDN Search: Open MSDN Help and locate to Search page.

 

MSDN Toolbar

MSDN Toolbar is as below. You can drop down to select searching history. Or enter word to search and press Enter or click 'Open MSDN Help in the IDE' button to search in MSDN.

 

Settings

MSDN Settings dialog is as below:

 

  • Toolbar

    • Show MSDN Toolbar: Show/Hide MSDN Toolbar in IDE.

    • Maximum History Items: Maximun searching history count in dropdown list.

  • Set MSDN

    • Default: Use auto-selected MSDN.

    • Use the following MSDN: Use selected installed MSDN.

    • Use MSDN Online: Use MSDN in Microsoft website.

    • Default URL: Set URL to default MSDN searching URL.

  • Other settings

    • Hotkey: Set the searching hotkey.
