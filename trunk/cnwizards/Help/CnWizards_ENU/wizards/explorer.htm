Explorer Wizard

This wizard can show a dockable Explorer window with files and directories list in IDE. It's as below:

The operations of this wizard are the same as those in Windows Explorer: Double clicking a directory to show its content in file list window. Double clicking a file to open it in windows. The 4 buttons' functions are:

  • Go to Up Folder: Go to the upper folder.

  • Arrange Icons: Choose the list style of files, such as Large Icon, Small Icon, List, Details.

  • Filter: File display filter can be set here.

  • Favorites: You can add Favorite folder to this menu to access them more freely.

  • New Folder: New a folder in selected folder.
