IDE External Wizard Management

This IDE External Wizard Management can manage external Wizards of IDE in Registry. Delphi/C++Builder/BDS supported.

The external Wizards registration information of Borland/CodeGear Delphi IDEs is all saved in Registry. This tool can manage them using visualization UI, and Enable/Disable feature is supported. Note: Only Win32 DLL Wizard supported, but not BPL/.NET Wizard.

The main dialog is shown as below: Left is IDE list. If an IDE is not installed, it will be shown as gray. When an IDE selected, its Wizard files will be listed. You can click the checkbox to enable/disable a wizard.

  • Reload: Discard changes and reload Wizard registration information of current IDE from Registry.

  • Save: Save all Wizard registration information in all IDE.

  • Add Wizard: Open a Wizard DLL file and register it to current IDE. Only Win32 DLL Wizard supported.

  • UnRegister Wizard: Remove registration information from current IDE, Keeping Wizard file unchanged.

  • Display File Properties: Display file properties dialog of this wizard file in explorer.

  • Open Directory: Open explorer and locate this wizard file.

  • Shell Menu: Popup Shell menu of this wizard file in explorer.

  • Select All: Enable all Wizards in current IDE.

  • Select None: Disable all Wizards in current IDE.

  • Select Inverse: Select Inverse in current IDE Wizard List.

  • Help: Show this Help.

  • Close: Close window and exit. If Wizard Registration Information changed, it will prompt to save.
