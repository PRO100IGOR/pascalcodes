Toggle Var Field

This tool is only valid in Delphi. Its default shortcut is Ctrl+Shift+V. When you pressing its shortcut or run it in menu item, cursor can jump from procedure to var field to let you add local var. If no "var" keyword exists, a "var" will be added automatically. Esc or pressing shortcut again to return.
