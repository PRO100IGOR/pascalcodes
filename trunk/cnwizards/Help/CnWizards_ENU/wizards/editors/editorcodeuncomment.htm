Uncomment Code Tool

This tool can search selected code block and remove '//' at linehead to remove the comment. Both Delphi and C++Builder are supported.

You may select code block before use this tool. If you select nothing, then current line will be processed. If the first linehead of selected code is not '//', then all lines will be ignored.
