Frame List

This function is the enhancement of "Select Frame to Insert" dialog in IDE. It can list all frames in current project or projectgroup to add to form. The information in list includes frame name, type, project, size and file type. Opened frame is in red. Quick search is supported.

This function will replace IDE's "Select Frame to Insert" dialog and be called to popup when user click the Frame button in component pallette and click form. Note: Whether replace(hook) or not is controlled together by the Hook button in "Use Unit" dialog of CnWizards for IDE's internal mechanism.

  • Add Selected Frame to Form: Add Selected Frame to Current Form.

  • Use Selected Unit: Use selected unit(s) for Current Unit in IDE.

  • Show Property of Selected Frame File: Show the file property dialog of selected frame file.

  • Copy Selected Frame Name to Clipboard: Copy selected Frame name to clipboard.

  • Match Frame Name Start: Match the frame name head when searching.

  • Match All Parts of Frame Name: Match any position of Frame name when searching.

  • Font: Change font of list.

  • Help: Show this Help.

  • Exit: Close this Window and Exit.
