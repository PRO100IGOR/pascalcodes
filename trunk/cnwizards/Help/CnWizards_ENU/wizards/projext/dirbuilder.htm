Project Dir Builder

This function allows you design directory structure for a project and create them in certain directory with Readme.txt files in it. Structure can be save to templates and load again.

The left panel is directory tree. Its checked state means whether create it. Yellow icon means there will be a Readme.txt file in it and blue means NOT. Directory node can be dragged freely.

  • Build Dir: Create directory structure from selected directory. The directory will be overwrited if exists before.

  • New Template: New a blank directory template.

  • Import Directory Structure: Import directory structure from an existing directory. See below.

  • Delete Template: Delete selected template in the dropdown list.

  • Save Template: Save current template, if it's a new, unnamed template, a dialog will appear to let you enter template name. If name exists, overwrite it.

  • Font: Change display font.

  • Help: Show this help.

  • Close Window: Close this window.

  • Template List: Show the templates in the dropdown list. If a template is selected, its content will be displayed.

  • Build Dir Description File: Whether create Readme.txt file in selected node's directory with the content in the memo.

 

Import Directories

"Import Directories" dialog is as below:

  • Dir: Select existing directory here.

  • Ignore this Dir: Ignore the directories with this name when importing. The name is NOT case sensitive and wildchar is not supported.

  • Auto Generate Readme File by Directory Name: Generate Readme.txt file in every node of structure. Each file's content is the directory name.

  • OK: Close this dialog and import directory structure.

  • Cancel: Close this dialog and do nothing.

  • Help: Show this help.
