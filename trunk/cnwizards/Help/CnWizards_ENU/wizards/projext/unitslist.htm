Unit List

This function can list all units in current project or projectgroup. The information in list includes name, type, project, size and file type. Opened file is in red. Quick search is supported.

  • Open Selected Unit: Open selected unit(s) in IDE.

  • Show Property of Selected Unit File: Show the file property dialog of selected file.

  • Copy Selected Unit Name to Clipboard: Copy selected unit name to clipboard.

  • Select All Units: Select all units in the list.

  • Cancel Selection: Select none in the list.

  • Inverse Selection: Inverse selection in the list.

  • Match Unit Name Start: Match the unit name head when searching.

  • Match All Parts of Unit Name: Match any position of unit name when searching.

  • Hook Project Unit List to IDE: Whether hook IDE's unit list action. If hooked, when 'Units...' was selected in 'View' menu, this dialog will appear.

  • Prompt when Open More than ONE Unit: Whether prompt for confirmation when open more than one unit.

  • Font: Change font of list.
