Historical Files Snapshot

This wizard can creat snapshots for often-used files to re-open it quickly. Its menu is as below:

Snapshots are displayed at the top of the sub items. Other functions related to snapshots and historical files are at the bottom of the separator line.

  • Create a Snapshot: Create a snapshot from all opened files. If this was clicked, a dialog will show to let user enter the snapshot's name. The created snapshot will be show in the menu items. You can click the item to open this snapshot.

  • Manage Snapshot List: List all snapshots and file list of each snapshot. You can delete snapshots, or add/edit/delete files of a snapshot, or change file orders of a snapshot.

  • Open Historical Files: Re-open historical files. Please refer to 'Open Historical Files'.
