Open High-Version Forms

This wizard is used to open high version forms in Delphi/BCB 5. Forms in Delphi/C++Builder 6 and above used UTF8 to save AnsiStrings and WideStrings, if DFM files includes SCB case characters, it can not opened in Delphi/BCB 5, even if saved to text format. This wizard can convert both text and binary format of high version forms.\nYou can use it to open high version forms directly

Noitce: corresponding units will be opened again, so remember to save the unit before re-opening if changes are made. Converted form is mandatory to text format.

Sometimes errors maybe occured that some properties can not be found because of incompatible of components. You can ignore them.

This wizard is only enabled in Delphi/BCB 5. If you only have high version IDE or need convert many files, you can use the DFM Convertor tool in CnPack IDE Wizards start menu.
