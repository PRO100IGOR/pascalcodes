Property Editors

Property Editors Introduction

There are some property editors in CnPack IDE Wizards to enhance Object Inspector in IDE.

The property editors can be configurated in Property & Component Editors Settings.

CnWizards' property editors support dynamic loading/unloading. They will take effects immediately after configuration without restarting IDE.

 

Boolean Property Editor

This editor can show a check box in Object Inspector to show its status.

 

Set Property Editor

IDE's default editor for set property only enables editing when expanding. This set editor can enables dropdown checking input and direct input of set property.

 

String & Caption Property Editor

This editor enables Multi-line text in Caption or other String properties.

Please refer to Multi-line String Editor.

 

Hint Editor

This editor enables Multi-line text and Long/short format hint in Hint property.

Please refer to Multi-line String Editor.

 

String List Editor

This editor provide enhanced functions for stringlist properties in IDE. The functions include searching, loading and saving.

Please refer to Multi-line String Editor.

 

FileName Editor

This editor allows user to select filename through an open-file dialog.

 

Constraints Editor

Editor for Constraints property.

Please refer to Constraints Editor.

 

Font Editor

This editor can shows font information with this font, instead of default 'TFont'.

 

Scrollbar Editor

This editor can show scrollbar's detailed information, including visible and position.

 

Align Editor

This editor shows a figure in Align property editor.

 

Component Name Editor

This editor can call Prefix Wizard to rename component names.
