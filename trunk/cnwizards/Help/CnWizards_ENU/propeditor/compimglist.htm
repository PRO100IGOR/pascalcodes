ImageList Component Editor

This component editor provides powerful design functions for ImageList component, including XP style icons with alpha channel, PNG support(auto converted to BMP), online searching, etc.

The UI of ImageList Component Editor is as below:

Selected Image

  • Preview Area: Show the preview image for selected one. The grid like a chessboard means transparent area.

  • Transparent Color: If selected image comes from PNG or BMP that added or searched, we can choose its transparent color with direct clicking using dropping button.

  • Use XP Style Image with Alpha channel: If checked, this ImageList will use 32 bit color format. It needs IDE to use XPManifest.

  • Options: Select crop or stretch or center mode if image size is not fit.

Images

  • Add: Open an external file and add to image list. File format can be BMP, ICO or PNG.

  • Replace: Open an external file to replace current selected image.

  • Delete: Delete selected image.

  • Clear: Clear all images in image list.

  • Export: Export selected image to a file. File format can be BMP or PNG.

  • Drag to Sort: You can drag images to adjust the sequence.

  • Select Image Size: Change the image size for ImageList.

Online Search

  • Keyword: Keyword to search. Currently only support English word, separated with space character.

  • Search: Perform Search using entered keyword.

  • Provider: Select icon provider.

  • Goto: Open selected provider website in browser.

  • Include only icons with commercial license.: Filter icon result with condition that whether allow commercial use.

  • Add: Add selected result icon to ImageList.

  • Replace: Replace selected image in ImageList with selected image in result.

  • Paging Buttons: Goto first, previous, next, last page in result icons.

  • MenuItem: Refresh: Refresh

  • MenuItem: Goto Page...: Goto page in result images.

  • MenuItem: Open Image in Browser...: Open browser to access this image from provider.

  • MenuItem: Search Iconset: Search iconset of current selected image. Only available for some particular icons.
