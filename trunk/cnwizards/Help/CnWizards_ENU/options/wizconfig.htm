CnWizards Settings

When user selects 'Settings...' in CnPack Menu, 'CnPack CnWizards Settings' dialog appears as below:

 

Wizards List (Left)

  • Name: Name of this Wizard.

  • State: Wizard's state: Active or Not.

  • Shortcut: Shortcut to call this wizard.

 

Wizard Info (Right)

  • Author: Author and contact info of this wizard.

  • Type: Wizard Type, including 'Common Wizard', 'Shortcut Wizard', 'Menu Wizard', 'Submenu Wizard', 'Repository Wizard', 'Unit Wizard', 'Form Wizard' and 'Project Wizard'.

  • Comments: Information about this Wizard.

  • Shortcut: You can specify the shortcut of current wizard. Note: Submenu wizard does NOT support shortcut.

  • Enabled: Set the wizard to enabled or disabled.

  • Settings: If current wizard supports settings, click this button will show its setting dialog.

  • Advanced: Advanced settings, see below.

 

Advanced Settings

Sort Menus

Click 'Advanced' button, a dialog shows as below:

All menu wizards' menu caption, wizard name, wizard ID and wizard type are listed out in 'Sort Menus' page. Click the column to sort menu items. Drag items to adjust the orders. Click 'Move Up' or 'Move Down' button to move selected item up or down. Click 'Reset' to reset menu items to original order. After adjustment, you can click OK to confirm, or Cancel to discard it.

Creation Settings

In 'Advanced' page, you can determine whether a wizard is created during start-up. It's different with enabled/disabled. Creation settings will take effects only after IDE restarting.

Click 'Select All' to select to create all wizards. And 'Unselect All' to select none. 'Reverse Select' to reverse selection of wizards.
