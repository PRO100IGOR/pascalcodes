Environment and Upgrade Settings

When user selects 'Settings...' in CnPack Menu, 'CnPack CnWizards Settings' dialog appears, the 'Environment Settings' page shows as below:

 

Hint

  • Show Hint Form for the First Time: Enables or disables the show of hint form for the first execute of wizard tool. If disabled, all hint forms will disappear. See Basic Operation.

  • Show Tooltips: Enables or disables the hint information of controls.

  • Reopen All Hint Forms: All settings of hint forms will reset after click this button. You will get prompt when you execute wizard tools next time.

 

Upgrade Settings

  • Disable Upgrade Prompt: Disables the feature of checking upgrade version when IDE startup.

  • Prompt when New Version Available: Enables the feature of checking upgrade version when IDE startup.

  • Ignore Beta Version: If checked, only prompt user when the new version is a release version.

  • Check for New Upgrade Now: Get the latest version information from server, include beta version.

See CnPack Upgrade Prompt.

 

Auxiliary Tools

  • Export IDE Image List: Save the IDE main imagelist to a folder, for developers.

  • Export IDE Action Names: Save the IDE main action names to a file, for developers.

  • Export Controls List: Save all installed package names and control names to Packages.txt and Components.txt file, for developers. Control names may duplicate since both of the VCL and CLX packages exist.

 

Others

  • Integrate CnPack Menu into Tools Menu: Put the 'CnPack' menu item into 'Tools' menu or in main menu.

  • Set IDE to run in CPU 0 single core: Used to fix some conflict under multi-core system.

  • Use SetThreadLocale to fix encode problem under Vista / Win7: Used to fix encode problem in some IDE.

  • Customize User Data Folder: Allow user to customize user data folder to avoid the authorization problem under Vista/Windows 7.
