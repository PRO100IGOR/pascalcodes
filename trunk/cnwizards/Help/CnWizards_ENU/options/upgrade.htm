CnPack IDE Wizards Upgrade Prompt

When the wizard finds new upgrading versions, a dialog shows as below:

You may perform these operations:

  • Download: Download the latest version from server. When the downloading finished, you can install it after closing the IDE.

  • Close: Close the dialog.

  • Never Prompt Me for this Version: If you do not want to upgrade to the new version, you can switch this on before you close the dialog, then you will be prompted until the next upgrade.

You can get more configurations information from Environment and Upgrade.
