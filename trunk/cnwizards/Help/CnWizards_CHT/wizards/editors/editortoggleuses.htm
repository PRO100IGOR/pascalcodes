切換引用區

此工具可供光標在當前位置和 uses/#include 之間跳轉，用於快速引用其他單元。默認快捷鍵是 Ctrl+Alt+U。對於 Pascal 文件，執行該工具可讓光標從當前位置跳至光標上部最近的 uses 部分，如第一次跳至 implementation 處的 uses 時，連續快速按兩次熱鍵能跳至 interface 處的 uses。對於 C 或 H 文件，可跳至頭部 #include 處。輸入完畢後再按一次快捷鍵或 ESC 返回。
