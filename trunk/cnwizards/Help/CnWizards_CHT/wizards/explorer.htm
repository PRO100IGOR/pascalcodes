文件管理器專家

該專家在 IDE 視窗中類比了 Windows 資源管理器的視窗，可顯示檔夾列表和檔列表，該專家的主窗口如下圖所示，可停靠在 IDE 中。

該專家的普通操作和 Windows 資源管理器的操作相同，雙擊目錄展開，雙擊檔則在 Windows 中打開。右面的四個按鈕功能如下：

  • 回上級目錄：去到上一層目錄。

  • 圖示排列方式：選擇檔管理器視窗中的圖示查看方式，如大圖示、小圖示、列表、詳細資料等。

  • 文件過濾：選擇在檔管理器視窗中顯示的檔的過濾方式，可自定義副檔名。

  • 文件夾收藏：可收藏一些常用的文件夾，以便在下拉功能表中直接查看。

  • 新建文件夾：可在當前選中的文件夾下新建一文件夾。
