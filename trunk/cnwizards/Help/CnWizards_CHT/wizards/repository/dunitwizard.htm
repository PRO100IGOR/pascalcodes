DUnit 測試用例生成嚮導

該嚮導可以生成 DUnit 的測試單元或工程，要求已經安裝 DUnit。執行此嚮導後，顯示的窗體如圖所示：

  • 新建 DUnit 測試工程：如果選項此項，將新建一 DUnit 測試 Application。

  • 新建 DUnit 測試單元：如果選項此項，則只新建一 DUnit 測試單元，測試用例的類名使用 TTest。

  • 初始化測試類：是否加入測試用例中 Setup 和 TearDown 過程的聲明和實現部分。

  • 加入單元頭註釋：是否在測試單元頭部加入註釋內容。

該專家只在 Delphi 下有效，並且在 BDS 中無效（BDS 已自帶 Unit Test 嚮導）。
