CPU 窗口擴展專家

CPU 視窗擴展專家為 CPU 視窗提供了彙編代碼複製功能，允許從彙編指令視窗中複製多行內容。

在 CPU 視窗彙編指令面板中點擊右鍵，彈出菜單如下圖所示：

    　

  • 複製30行到剪貼板：從彙編代碼面板中複製指定行數的內容到剪貼板或檔，該功能表執行的操作可能受後面設置的影響。

  • 複製代碼...：自定義方式從彙編代碼面板中複製代碼。

執行“複製代碼”後，顯示如下圖所示視窗。

　

  • 複製行數：要複製的彙編代碼行數。

  • 從第一行：從彙編代碼面板顯示的第一行開始複製。

  • 從選擇行：從當前選擇的代碼行開始複製。

  • 複製到剪貼板：複製結果放到剪貼板。

  • 複製到檔：複製結果保存到檔中。

  • 應用設置為默認操作：如果選擇該項，當前設置的複製方式會自動應用到前述第一個功能表項。

  • 確定：如果該視窗在“複製代碼...”功能表項被調用，將執行代碼複製操作。如果在設置介面中調用，將保存當前參數設置。

  • 取消：放棄當前操作。
