版本信息擴展專家

版本信息擴展專家能夠在編譯時自動升級工程的版本號，並且能夠將編譯時間插入版本信息中。該專家的功能僅在特定版本的 IDE 中有效。當前不支持 Delphi 5 和 C++Builder 5/6。

該專家的設置窗口如下圖所示：

  • 版本信息中插入編譯時刻：在版本信息中插入最後一次編譯的時刻。此功能依賴於工程選項的版本信息列表中是否支持編譯時間（LastCompiledTime）這一項，因而可能在部分 IDE 下無效。

  • 編譯時自動增加 Build 號：在每一次編譯時（而不僅僅是 Build 時）將 Build 版本號加一。

註：以上兩選項只在工程選項中包含版本信息時有效，選項頁中「Include version information in project」未選中時無效。
