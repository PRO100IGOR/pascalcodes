Repository 专家

CnPack IDE Wizards 自带了一些 Repository 专家，也就是用来生成特定的新项目或单元的向导。可以在 IDE 的 File 菜单下的 New...对话框中的 CnPack 页中访问到它们。Repository 专家也在不断的开发中，您看到的向导可能比现在要多。

另外，在 CnPack 菜单中的 “Repository 列表”一项中，也可以通过点击其子菜单来启动各个 CnPack 专家包自带的 Repository 专家。
