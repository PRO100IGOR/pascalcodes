代码窗口全屏切换工具

该工具允许用户快速在全屏方式和普通方式中切换代码编辑器窗口显示模式。

IDE 在启动后，虽然其代码窗口状态为最大化，但事实上并没有最大化。如果选择“IDE 启动后自动将代码窗口最大化”，该工具会在 IDE 启动后自动将代码窗口最大化。

当代码编辑器窗口最大化显示时，如果选择了“自动隐藏主窗体”，则 IDE 主窗体会收缩成屏幕顶部的横条。鼠标移动到屏幕顶部自动显示完整的 IDE 主窗体，移出窗体时自动隐藏。该功能只有在代码编辑器窗口设置为最大化显示、窗口状态为最大化，且 IDE 窗体未最大化时有效。
