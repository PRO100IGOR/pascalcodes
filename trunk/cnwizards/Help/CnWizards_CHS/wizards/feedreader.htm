源阅读专家

该专家用来订阅 RSS 和 ATOM 源，并将内容显示在 IDE 的状态栏上面。

源内容显示

源标题将动态显示在 IDE 下方的状态栏上并显示为超链接，点击可打开相关页面。

鼠标移入状态栏后，在右方会显示三个按钮，分别是：上一个、设置、下一个。

右键点击可显示菜单。

  • 上一条：切换到上一条内容。

  • 下一条：切换到下一条内容。

  • 全部更新：更新当前订阅的所有频道。

  • 关闭：关闭专家。

  • 设置：打开设置对话框。

　

源阅读专家设置

设置对话框如图所示：

　

  • 订阅 CnPack 频道：是否订阅 CnPack 官方频道，建议启用。

  • 订阅 CnPack 合作伙伴频道：是否订阅 CnPack 合作伙伴官方频道，建议启用。

  • 随机：是否随机显示源内容。

  • 切换周期：自动切换源条目的周期。

  • 源列表：自定义的源列表，支持 RSS 和 ATOM 格式。

    • 增加：增加一个新的源。

    • 删除：删除当前选择的源。

    • 标题：标识当前源，只用于显示。

    • 链接：源链接，支持 RSS 和 ATOM 格式。

    • 更新周期：设置自动更新源内容的时间：

    • 限制：限制只显示最近的条目数量，0为不限制。

  • 确定：保存修改，关闭对话框。

  • 取消：放弃修改，关闭对话框。

  • 帮助：显示本帮助。
