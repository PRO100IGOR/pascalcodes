Frame 列表

本功能是 IDE 中插入 Frame 对话框功能的扩展。本功能可显示当前工程中可供插入的所有 Frame 的名称、所属工程名、大小和文件类型等，可根据文件名查找过滤，其中红色为已打开项目。

本功能在当点击组件面板上的 Frame 图标并在当前窗体上点击时被调用，取代了 IDE 本身的插入 Frame 对话框。由于 IDE 本身的特性，本取代功能是否使能由“引用单元列表”对话框中的挂接按钮统一控制，使用时请注意。

  • 将所选 Frame 加入窗体：将所选的 Frame 加入当前窗体。

  • 显示所选 Frame 的文件属性：显示所选择的 Frame 所在的单元文件的属性对话框。

  • 复制 Frame 名称：复制所选择的 Frame 名称到剪贴板。

  • 按Frame 名称开头匹配：查找的时候，输入的字符从 Frame 名称的头部开始匹配，单元名头部含有输入字符的才能显示。

  • 按 Frame 名称的所有位置匹配：查找的时候，输入的字符匹配 Frame 名称的任何部分，单元名含有输入字符的才能显示。

  • 字体：更改列表框字体。

  • 帮助：显示此帮助。

  • 关闭：关闭此窗口。
