IDE 专家管理工具

该专家专家管理工具通过注册表进行 IDE 的外部专家管理，支持 Delphi/C++Builder/BDS。

Borland/CodeGear Delphi 系列 IDE 的外部专家注册信息均保存在注册表中，本工具能以可视化的方式提供对注册表中专家注册信息的管理，包括启用、禁用等。另外本专家只支持 Win32 类型的 DLL 专家，不支持 BPL 以及 .NET 类型的专家。

该工具主界面如下图所示：左面列出了支持的 IDE 版本类型，未安装的 IDE 版本以灰色显示。当选中一 IDE 类型时，右面会出现其已安装的 DLL 专家列表，用户可通过点击列表框中的复选框方便地启用、禁用它们。

  • 重新载入：从注册表中载入选中 IDE 类型的专家信息，界面上的改动将被忽略。

  • 全部保存：保存全部 IDE 中的已改动的专家注册信息。

  • 新增专家：打开一专家文件并注册到当前 IDE 中只支持 Win32 DLL 类型的专家文件。

  • 删除专家：从注册表中彻底删除此专家的注册信息，专家文件本身不受影响。

  • 显示专家文件属性：显示资源管理器中的该专家的文件属性对话框。

  • 打开所在目录：在资源管理器中打开该专家文件所在目录并定位到此专家文件。

  • 外壳关联菜单：显示资源管理器中此专家文件的外壳关联右键菜单。

  • 全选：全部启用当前 IDE 中的专家。

  • 全不选：全部禁用当前 IDE 中的专家。

  • 反向选择：反向选择当前 IDE 专家列表中的专家。

  • 帮助：显示此帮助内容。

  • 关闭：关闭程序退出。如果有 IDE 改动过，则提示保存。
