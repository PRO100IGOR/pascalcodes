CnMemProf 工程向导

该向导可自动生成一引用 CnMemProf.pas 单元来实现内存防护功能的工程，能够设定是否记录对象信息，是否记录日志文件等选项。此向导生成的工程文件需要 CnMemProf.pas 才能正常编译运行，CnMemProf.pas在专家包安装目录下的 Source 目录中。执行此向导后，显示的窗体如图所示：

  • 程序结束后弹出内存报告消息框：勾上此选项后，程序运行退出时，CnMemProf 单元通过弹出消息框的方式来报告内存使用情况。

  • 记录对象列表：勾上此选项后，CnMemProf 单元会对程序中生成的各个对象加以记录。

  • 使用 RTTI 记录对象类型信息：勾上此选项后，CnMemProf 单元将使用 RTTI 记录程序中生成的各个对象的类型信息。

  • 将内存使用情况记录到日志文件：勾上此选项后，CnMemProf 单元会将内存使用情况记录到日志文件中。

  • 日志文件名：选择保存的日志文件名，如为空，默认为程序同一目录下的 Memory.log 文件。

该专家只在 Delphi 环境下有效。
