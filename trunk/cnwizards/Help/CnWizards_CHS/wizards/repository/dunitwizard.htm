DUnit 测试用例生成向导

该向导可以生成 DUnit 的测试单元或工程，要求已经安装 DUnit。执行此向导后，显示的窗体如图所示：

  • 新建 DUnit 测试工程：如果选项此项，将新建一 DUnit 测试 Application。

  • 新建 DUnit 测试单元：如果选项此项，则只新建一 DUnit 测试单元，测试用例的类名使用 TTest。

  • 初始化测试类：是否加入测试用例中 Setup 和 TearDown 过程的声明和实现部分。

  • 加入单元头注释：是否在测试单元头部加入注释内容。

该专家只在 Delphi 下有效，并且在 BDS 中无效（BDS 已自带 Unit Test 向导）。
