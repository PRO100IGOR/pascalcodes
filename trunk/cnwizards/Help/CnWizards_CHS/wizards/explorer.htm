文件管理器专家

该专家在 IDE 窗口中模拟了 Windows 资源管理器的窗口，可显示文件夹列表和文件列表，该专家的主窗口如下图所示，可停靠在 IDE 中。

该专家的普通操作和 Windows 资源管理器的操作相同，双击目录展开，双击文件则在 Windows 中打开。右面的四个按钮功能如下：

  • 回上级目录：去到上一层目录。

  • 图标排列方式：选择文件管理器窗口中的图标查看方式，如大图标、小图标、列表、详细资料等。

  • 文件过滤：选择在文件管理器窗口中显示的文件的过滤方式，可自定义扩展名。

  • 文件夹收藏：可收藏一些常用的文件夹，以便在下拉菜单中直接查看。

  • 新建文件夹：可在当前选中的文件夹下新建一文件夹。
