ImageList 组件编辑器

该编辑器为 ImageList 提供了强大的设计支持，包括 XP 风格带 Alpha 通道的图像列表支持、png 文件支持（自动转换为 Bitmap）和在线搜索图标等功能。

下图为 ImageList 组件编辑器界面：

选中的图像

  • 图像预览：预览当前选中的图像，背景棋盘格表示图像的透明部分。

  • 透明色：如果选中的图像是通过文件或搜索添加的 png 和 bmp 文件，可以选择其透明色。点击下拉框右边的按钮，可从图像预览区点取透明色。

  • 使用带透明通道的 XP 风格图标：选中该选项后，图像列表将使用 32 bit 的格式。该选项需要 IDE 本身支持 XPManifest 才会启用。

  • 选项：如果选中的图像是通过文件或搜索添加的 png 和 bmp 文件，设定显示的缩放方式。

图像列表

  • 增加：从文件系统中选择一个或多个图像文件，添加到列表中。支持 bmp、ico 和 png 三种格式。

  • 替换：从文件系统中选择一个或多个图像文件，替换掉当前选中的图像。

  • 删除：删除当前选中的图像。

  • 清空：清空当前图像列表。

  • 导出：导出当前选中的图像或全部图像为图像文件，支持 bmp 和 png 格式导出。

  • 拖动排序：在图像列表中，可通过选中图标拖动来进行排序。

  • 图像大小：可以快速修改图像列表的大小。如果当前图像列表不为空，修改大小后会自动对图像进行缩放。

在线搜索

  • 关键字：输入要搜索的关键字，多个关键字用空格隔开，目前只支持英文单词。

  • 搜索：开始搜索图标。

  • 图标来源：选择提供图标搜索服务的网站。

  • 转到：在浏览器中打开当前选择的图标搜索服务网站。

  • 只包含允许商业授权的图标：搜索图标时指定是否在结果中只包含允许在商业软件中使用的图标。

  • 增加：将当前选中的图像添加到左边的图像列表中。

  • 替换：用当前选中的图像替换左边的图像列表中选中的图像。

  • 翻页：在搜索结果中翻页。

  • 刷新菜单：刷新当前页面的显示。

  • 转到页数：跳转到指定页数显示。

  • 在浏览器中打开图像：直接从网站上访问当前选中图像。

  • 搜索图标集：搜索当前选中图像所从属的图标集，该功能只对部分图标来源有效。
