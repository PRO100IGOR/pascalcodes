CPU 窗口扩展专家

CPU 窗口扩展专家为 CPU 窗口提供了汇编代码复制功能，允许从汇编指令窗口中复制多行内容。

在 CPU 窗口汇编指令面板中点击右键，弹出菜单如下图所示：

    　

  • 复制30行到剪贴板：从汇编代码面板中复制指定行数的内容到剪贴板或文件，该菜单执行的操作可能受后面设置的影响。

  • 复制代码...：自定义方式从汇编代码面板中复制代码。

执行“复制代码”后，显示如下图所示窗口。

　

  • 复制行数：要复制的汇编代码行数。

  • 从第一行：从汇编代码面板显示的第一行开始复制。

  • 从选择行：从当前选择的代码行开始复制。

  • 复制到剪贴板：复制结果放到剪贴板。

  • 复制到文件：复制结果保存到文件中。

  • 应用设置为默认操作：如果选择该项，当前设置的复制方式会自动应用到前述第一个菜单项。

  • 确定：如果该窗口在“复制代码...”菜单项被调用，将执行代码复制操作。如果在设置界面中调用，将保存当前参数设置。

  • 取消：放弃当前操作。
