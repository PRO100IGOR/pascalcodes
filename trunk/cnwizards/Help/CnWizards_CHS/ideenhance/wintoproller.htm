窗口置顶与折叠扩展专家

窗口置顶与折叠扩展专家能为 IDE 中的非模态窗口的标题栏增加置顶和折叠等按钮（折叠按钮的功能是将窗口缩小至标题栏大小以少占用桌面空间），默认会增加三个按钮：选项下拉菜单按钮、折叠按钮和置顶按钮。下图是 Object Inspector 增加标题栏按钮后的效果：

  • 选项下拉菜单：点击此按钮后弹出有关于标题栏按钮的下拉菜单。

  • 折叠按钮：点击此按钮后，窗口会折叠缩小至标题栏大小，再点击一次还原。部分窗体可能因为本身设置了 AutoSize 或 Constraints 属性导致不能缩小至标题栏大小，使用时请注意。

  • 置顶按钮：点击此按钮后本窗口将置顶，再点击一次还原。

  • 将此窗体加入屏蔽列表：本专家允许用户设置根据窗体类名来决定是否显示标题栏按钮的过滤器，本菜单项的功能是将本窗体的类名加入过滤列表以阻止显示标题栏按钮。

  • 设置：点击此按钮后将弹出设置窗口。

本专家的设置窗口如下图所示：

　

  • 按钮类型：控制在标题栏中增加的按钮类型，包括是否显示置顶、折叠、设置按钮等。

  • 忽略标题栏文字宽度：如不勾上此选项，按钮将显示在标题栏文字之后，如窗体标题栏文字过长，按钮则不显示。勾上此选项后将忽略标题栏文字的长度，而总在右边显示标题栏按钮。

  • 窗口折叠使用动画效果：控制是否在窗口的折叠过程中使用动画缩进展开效果。

  • 启用自定义窗体类名过滤以阻止部分窗口显示扩展按钮：部分非模态窗体是不需要显示置顶等按钮的，如 IDE 主窗体等，而且用户也可能不需要某些类型的窗体显示其标题栏按钮。这里提供了根据窗体类名的过滤功能，凡是在列表中并且勾上的过滤规则都会生效，和规则中标识的窗体类名相同的窗体，都不会显示标题栏按钮。两下拉框的下拉列表里预置了 IDE 中常见的窗口类名和其注释，列表如下：

  • 替换：将两个下拉框中编辑的规则类名和注释替换选中的规则。

  • 添加：将两个下拉框中编辑的规则类名和注释作为一条新规则加入到规则列表中。

  • 删除：删除规则列表中的选中规则。

  • 确定：关闭此窗口，保存选项设置内容并更新生效。

  • 取消：关闭此窗口，忽略修改的选项内容。

  • 帮助：显示帮助信息。
