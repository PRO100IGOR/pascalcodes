Tools 辅助工具目录
=================

该目录存放由CnPack开发组编写的辅助工具源码。

CnCleanIDE(-)
------
类型：专家包工具
功能：IDE 历史记录清理工具
版本：V1.0
使用：直接运行使用。
作者：刘啸（LiuXiao） liuxiao@cnpack.org
时间：2005.02.20
备注：此功能已整合入 IDE 辅助备份/恢复工具，故不再使用。

CnConfigIO
------
类型：专家包工具
功能：专家包设置导入导出工具
版本：V1.0
使用：直接运行使用。
作者：刘啸（LiuXiao） liuxiao@cnpack.org
时间：2003.05.20

CnDebugViewer
-------------
类型：专家包工具
功能：调试信息查看工具
版本：V1.4
使用：直接运行使用。
作者：刘啸（LiuXiao） liuxiao@cnpack.org
时间：2006.04

CnDfm6To5
---------
类型：专家包工具
功能：将D6/7下的DFM文件转换为D5兼容的格式
版本：V1.0
使用：直接运行使用，支持文件和目录转换。
作者：周劲羽 (zjy@cnpack.org)
时间：2003.04.03

CnIdeBRTool
-----------
类型：专家包工具
功能：IDE 辅助备份/恢复工具
版本：V1.0
使用：直接运行使用。
作者：ccRun（老妖）
移植：刘啸（LiuXiao） liuxiao@cnpack.org
时间：2006.08.23
备注：此功能已整合了 IDE 打开文件历史记录清除工具。

CnWizRes
--------
类型：专家包工具
功能：将专家包使用到的图标转换成资源文件
版本：V1.0
使用：直接运行加参数使用。
作者：周劲羽 (zjy@cnpack.org)
时间：2006.03

CnWizSetup
----------
类型：辅助安装
功能：安装/反安装CnWizards IDE专家包，支持D5/6/7、BCB5/6
版本：V1.1
使用：第一次运行安装专家包，再次运行反安装，也可带参数 /i 进行安装，/u 反安装。
作者：周劲羽 (zjy@cnpack.org)
时间：2002.10.03
