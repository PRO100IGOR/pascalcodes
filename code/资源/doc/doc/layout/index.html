    平台封装

    1:以上文件中可能有的效果出不来，但是在平台中是好的，因此只需要看他们的属性即可。。
    2:基本上所有的表单元素都扩展了一个refresh方法，比如官方的文本框中，动态创建的不支持效果，但是在平台中调用按钮的refresh方法就支持了。
    另外，官方中所有刷新的方法，如单选、多选、下拉等选中效果变化时都调用refresh方法可以实现刷新。
    3:官方api中特殊的属性，如文本框的水印属性，平台对struts的html:text进行了扩展，html:text已经支持水印效果了。
    4:平台对html:button进行了扩展，原来必填的prototype属性不必填了，增加非必填属性buttontype,就是按钮api中的按钮的样式:icon_add中的add,如果 不写styleid，那么id就是add
    例如 buttontype=add,则会出现 button id="add" span class="icon_add" 保存 的按钮 5:下拉框、多选、单选支持原生的onchange、onclick事件了，而且可以用this，但是下拉框要在事件中判断this.value，如果是0，表示是无效事件调用，一定要注意！
