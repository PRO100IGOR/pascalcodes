参数名默认值参数说明

			
			
				
width400面板宽度，也可以使用百分比，如90%。注意数字不加引号，百分比加引号。另外当direction为"ml"或"mr"时不可用百分比。无height参数，因为面板高度自适应内容。

				
beforeClickText"打开面板"面板展开之前的按钮上的文字

				
afterClickText"关闭面板"面板展开之后的按钮上的文字

				
duration500面板展开动画持续时间，单位毫秒

				
animatefirsttrue初始时是否以动画方式收缩

				
direction"tc"面板放置的方位。有8个可选项，tl：居上靠左（top,left）；tc：居上居中（top,center）；tr：居上靠右（top,right）；ml：中间靠左（middle,left）；mr：中间靠右（middle,right）；bl：下方靠左（bottom,left）；bc：下方居中（bottom,center）；br：下方靠右（bottom,right）；

				
iframefalse此参数默认为false表示将本页中的一个div里的内容作为浮动面板的内容，如果想使用某一页的内容作为浮动面板的内容，则将该页路径赋予iframe属性，如iframe:"27-2.html"。同一个网页不能有两个以上的iframe模式的面板，而且iframe模式的面板必须要在最后定义

				
depth600++面板的层级深度，生成第一个面板的层级默认为700，一会每次生成一个面板会+1。如果有多个面板并且可能会相互叠加，那么depth大的会把小的盖住（即后生成的会盖住之前生成的）。通过设置depth可以自定义各面板层次关系。不建议小于700否则可能无法遮挡住select下拉框。

				
init"hide"初始时是打开状态还是关闭状态。为"show"时初始为打开状态，这时需要自定义beforeClickText和afterClickText，因为默认的文字是相反的。这时如果animatefirst为true时初始时会以动画形式出现。

				
topDistance30面板距顶部的位置。当direction为ml或mr时有效
