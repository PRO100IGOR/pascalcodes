  1. 开始

    建表

  2. 引用

    引入applicationContext-formrecord.xml、struts-config-formrecord.xml、Formrecord.hbm.xml,并在web.xml中配置struts-config-formrecord.xml

  3. 调用

    sihe:log 标签

	参数说明：
	formName ：* 表单NAME
	explain ：设置表单字段的描述名称，优先获取DOM元素的alt属性(如 [html:text alt="名字" prototype="name"])，
	否则读取window.${formName}Explain对象，该对象的格式是：{"字段名称1":"描述名称1","字段名称2":"描述名称2"...}，如果连这个对象也没有，则描述名称是控件name属性。
	ignoreExpr 忽略字段的jquery选择字符串，如所有忽略字段加一个class not，则字符串是 .not
	ignoreHidden：是否忽略隐藏域，默认为true,ignoreExpr优先级高于ignoreHidden
	handle：监视表单字段的动作，默认为update,可选值“add|delete|update”
    注意：1：由于部分表单页面在页面加载完毕时会对字段用js赋值，而本组件必须获取到字段的初始值才能进行比对，所以尽量将本组件放在页面的最底部，这样能保证组件获取字段值是在页面加载完毕并js赋值完毕后才获取字段值。
