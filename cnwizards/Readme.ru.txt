1. Введение в CnPack IDE-Мастер
=====================================================

    CnPack IDE-Мастер - флагманский продукт CnPack. Полное имя на русском 
'CnPack IDE-Мастер'. Это бесплатный набор плагинов для Delphi/
C++Builder/BDS для повышения эффективности разработки.

    CnPack IDE-Мастер предназначен для разработчиков, использующих 
Delphi 5 6 7, C++Builder 5 6 и BDS 2005 2006, RAD Studio 2007 2009 2010, XE, XE 2.

    Официальная версия CnPack IDE-Мастера включает следующие языки: 
Китайский (упрощенный и традиционный), Английский, Русский и Немецкий.

4. Структура
======================================================================

    Структура проекта CnPack IDE-Мастера следующая:

    1. Bin. Откомпилированные и другие файлы проета, включая языковые.
    2. Dcu. Временная папка для компиляции.
    3. Doc. Документация CnPack IDE-Мастера.
    4. Help. HTML-файлы справочной системы CnPack IDE-Мастера.
    5. Install. Установочные скрипты.
    6. Make. Скрипты автоматической компиляции.
    7. Source. Исходный код CnPack IDE-Мастера.
    8. Tools. Исходный код утилит, входящих в состав CnPack IDE-Мастера.

6. Компиляция и сборка
======================================================================

    Для начала Вам необходимо загрузить исходный код CnPack и извлечь его
в папку, где находится CnPack IDE-Мастер. Далее откройте проект в папке
"Source" с откомпилируйте его. Папка "Tools" содержит исходный код других
утилит CnPack IDE-Мастера.

    Утилита для авто-компиляции находится в папке "Make". Вы можете
запустить "want.exe" из командной строки для получения справки.
