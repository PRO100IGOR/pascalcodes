Tabset Enhancement

IDE's source code tabset has nothing to indicate whether an opened file was modified. This tool can add a "*" character to all modified files after their caption.

The new tabset shows as below:

     

If a unit associates with a form, when the form modified, the unit tab also shows the "*" character even the source is not modified.

The settings dialog is as below:

 

  • Add "*" to Modified File's Name in Tabset: Enable or disable this feature. Note: The "*" feature is disabled in BDS, for BDS has a Tab Icon Changing mechanism in itself.

  • Close Page by Double-click: Whether allows closing an editor page by double-click its tab.

  • Set Editor Tab to Multi-line: Whether Set Editor Tab to Multi-line Style. It's not applicable for BDS.

  • Set Editor Tab's Style to Flat Button: Whether Set Editor Tab to Flat Button Style. It's not applicable for BDS.

  • Show Shell Menu by Right-click Tabset with Shift or Ctrl: Whether allows showing shell context menu by right-click with shift or ctrl key.
