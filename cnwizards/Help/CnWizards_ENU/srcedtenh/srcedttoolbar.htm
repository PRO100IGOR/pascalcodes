Editor Toolbar

This tool adds a toolbar in IDE's code editor, and you can customize its buttons. It is very useful when code editor is zoomed to full screen.

The Editor Toolbar shows as below:

Right click the toolbar and select Customize menu, the customize dialog shows as below:

  • Category: Display all categories of buttons in IDE.

  • Available Buttons: Display all available buttons in current category.

  • Toolbar Buttons: Display all buttons to be placed on editor toolbar.

  • Add/Delete/Reset: Used to adjust and reset the buttons on editor toolbar.

  • Add Separator: Add a separator button in toolbar.

  • OK: Save the settings.

  • Cancel: Cancel the settings.

  • Help: Show this help information.

 

Toolbar settings dialog is as below:

  • Show Toolbar in Editor: Whether show the toolbar in code editor.

  • Show Toolbar in BDS Embedded Designer: Whether show the toolbar in BDS Embedded VCL Designer. Only enabled in BDS.

  • Customize Buttons: Customize toolbar buttons, see before.

  • Wrap Buttons: Whether automatically wrap buttons when toolbar can not display all buttons in single line.
