Auto Save

This function can save all opened files in editor timely. The default saving interval is 2 minutes. It's defaultly disabled to avoid saving unnecessary contents.

The setting dialog is shown as below:

     

  • Auto Save All: If checked, All opened files will be auto saved timely. The interval is specified by the interval minutes. But those files, unchanged or not exists in disk, will not be saved automatically.
