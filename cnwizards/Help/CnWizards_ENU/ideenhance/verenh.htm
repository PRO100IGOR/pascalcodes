Version Enhancements Wizard

This wizard can auto-increment version number and insert compiling time into version info. Currently Delphi 5 and C++Builder 5/6 are NOT supported.

The Setting dialog is shown as below:

  • Insert Compiling Time into Version Info: Insert the last compiling time into project version info. This function depends on whether IDE supports LastCompiledTime key in version info.

  • Auto-increment Build Number when Compiling: Increment Build Number when not only every building, but also every compiling.

Note: Both are only enabled when "Include version information in project" was checked in Project Options dialog.
