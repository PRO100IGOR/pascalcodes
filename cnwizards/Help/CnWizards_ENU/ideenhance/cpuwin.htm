CPU Window Enhancements Wizard

CPU Window Enhancements Wizard provides a feature of copying ASM code in CPU window. The wizard allows user to copy multi-line ASM code once.

Right click in the ASM panel of CPU window, the context menu shows as below:

     

  • Copy 30 Lines: Copy given code lines to clipboard or file from ASM panel, affected by configuration.

  • Copy ASM Code: Copy code from ASM panel in user-defined mode.

When execute 'Copy ASM Code' the dialog shows as below:

 

  • Copy Lines: The number of to copy.

  • From Beginning: Copy code from the first line of ASM panel displayed.

  • From Selected Line: Copy code from current line.

  • Copy to Clipboard: The result of copying will put to clipboard.

  • Copy to File: The result of copying will save as a file.

  • Set as Default: If checked, the settings will apply to the first menu referenced before automatic.

  • OK: If the dialog is called by 'Copy ASM Code' menu, the copy operation will execute. If it is called from settings dialog, then saves the settings.

  • Cancel: Cancel the settings.
