Property and Component Editors Settings

When user selects 'Settings...' in CnPack Menu, 'CnPack CnWizards Settings' dialog appears, the 'Property Editors' page shows as below. Each property should be in one line.

 

Editor List (Left)

  • Name: The name of this Property/Component Editor.

  • State: Editor's state: Active or Not.

 

Editor Info (Right)

  • Author: Author and contact info of this editor.

  • Type: Editor Type, including 'Property Editor' and 'Component Editor'.

  • Comments: Information about this editor.

  • Enabled: Switch the editor to enabled or disabled.
