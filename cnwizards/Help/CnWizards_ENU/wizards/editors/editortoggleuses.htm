Toggle Uses/Include Field

This tool is used to jump between current place and uses/include part in source file. When you pressing its shortcut, which default is Ctrl+Alt+U, Cursor can jump from current place to upper uses (Pascal file) or #include(C file). If it's implementation uses, you can jump to interface uses by pressing its shortcut twice quickly in Pascal file. Esc or pressiong shortcut again to return.
