Delete Blank Lines Tool

This tool can delete the blank lines in selected block or current file.

  • Context to Process: Select the context to process is selected block or current file.

  • Compact Continuous Blank Lines to One: If select this, the tool will search all continuous blank lines and compact them to one line. Single blank line will be ignored.

  • Delete All Blank Lines: If select this, then all blank lines will be deleted.
