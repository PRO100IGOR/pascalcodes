Code Editor Wizard

Code Editor Wizard is designed to extend code editor. It includes a group of additional tools, Both Delphi and C++Builder are supported.

The settings dialog shows as below:

Left area lists all tools and their information.

Right panel is used to setting their shortcut, and to enable or disable them.

Bottom area displays current tool's simple introduction.
