Editor Fullscreen Switch

This simple tool allows user to switch code editor window between fullscreen mode and normal mode.

You can check "Maximized when IDE Start" to set code editor window to fullscreen mode at IDE starting.

When editor window is fullscreen, if you select "Auto hide main form", then IDE's main form will shrink to a line on the top of screen. Move mouse enter this line, the form will auto display, and it will auto hide when mouse leave it. This feature only available when editor window is maximized and fullscreen and IDE's main form is not maximized.
