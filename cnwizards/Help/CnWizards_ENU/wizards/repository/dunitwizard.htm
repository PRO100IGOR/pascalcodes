DUnit Test Case

This wizard can generate DUnit test case unit or project. It requires DUnit. The dialog is as below:

  • New DUnit Test Application: New a DUnit Test Application project if this was selected.

  • New DUnit Test Unit: New a DUnit Test unit if this was selected.

  • Initialize Test Case: Whether add the 2 method: Setup and TearDown into unit.

  • Add Comment Head: Whether add comment head to unit.

Only Delphi is supported. And BDS not supported, for BDS has a Unit Test wizard already.
