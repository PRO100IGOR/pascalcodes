Clean Temporary Files

This function can clean temporary files of certain types in project directory. Its dialog is as below:

  • Path: You can select and edit the directories here. If the checkbox was selected, this directory and its sub-directories will be searched, otherwise only this directory will be searched. Right-click to popup a menu to Select All, Unselect All and Inverse Selection.

  • File Type: Here list the file types need to be deleted. You can add customized extensions or delete some to do not clean these types. Right-click to popup a menu to Select All, Unselect All and Inverse Selection.

  • Keep Obj/Dcu Files when NO Same Source Files Exist: If checked, the dcu/obj files without corresponding source files such as pas/cpp/c will be kept and not deleted. It's used to avoid deleting binary libraries without source code.

  • Default: Restore file types to default setting, which includes most temporary types.

  • Cancel: Close this dialog without cleaning.

  • Previous:Goto previous page to display this setting.

  • Next: Search and display the files. You can ignore some files in it.

  • Finish: Search and delete found files directly without prompting. If cleaning succeed, file count and all bytes will be reported.
