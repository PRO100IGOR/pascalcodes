Form List

This function can list all forms in current project or projectgroup. The information in list includes name, type, project, size and file type. Opened file is in red. Quick search is supported.

  • Open Selected Form: Open selected form(s) in IDE.

  • Convert Selected Binary Form to Text Form: Convert selected form from binary format to text format.

  • Convert Selected Text Form to Binary Form: Convert selected form from text format to binary format.

  • Show Property of Selected Form File: Show the file property dialog of selected form file.

  • Copy Name of Selected Form to Clipboard: Copy Name of Selected Form to Clipboard.

  • Select All Forms: Select all forms in the list.

  • Cancel Selection: Select none in the list.

  • Inverse Selection: Inverse selection in the list.

  • Match Form Name Start: Match the form name head when searching.

  • Match All Parts of Form Name: Match any position of form name when searching.

  • Hook Project Form List to IDE: Whether hook IDE's form list action. If hooked, when 'Forms...' was selected in 'View' menu, this dialog will appear.

  • Prompt when Open More than ONE Form: Whether prompt for confirmation when open more than one form.

  • Font: Change font of list.

  • Help: Show this help.

  • Exit: Close this window and Exit.
