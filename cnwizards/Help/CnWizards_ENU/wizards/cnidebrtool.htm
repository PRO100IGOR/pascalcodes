IDE Config Backup Restore Tool

This wizard, which can also run in stand-alone mode, is used to backup/restore IDE Configurations of Delphi/C++Builder/BDS.

It's Inconvenient for Our Delphi/C++Builder Users to Re-config IDE Settings Manually after Re-install IDE. And What a Pity if Our Valuable Settings Lost! This Tool can Help us to Backup Delphi/C++Builder IDE Settings Information, including Code Templates, Menu Templates, IDE Environment and Editor Configuration and IDE Desktop Settings. Now Delphi 5/6/7/8, C++Builder 5/6 and BDS 2005/2006, RAD Studio 2007/2009 Supported. And, IDE History Cleaner also included Here.

 

Backup IDE Settings

If select "Backup IDE Settings" and click Next, now you need to select an installed IDE. This tool only can backup one IDE settings at a time.

Select IDE and click Next. Now you need to select what to backup. They include:

  • Code Templates: Code Templates defined in Editor Options.

  • Object Repository: Objects in New dialog, including Forms, Units or Projects.

  • IDE Configuration: IDE Configuration in Registry, including Window Layout, Environment Options and Editor Options, etc.

  • Menu Templates: Menu Templates in Menu Designer.

Check the boxes to backup and select a file and click Next, selected contents will be backuped to this file. The backup file, with "bic" extension, is CnWizards' customized format. You can keep this file for future usage.

 

Restore IDE Settings

Click Next, you can see the installed IDE directory. And you can select what to restore as below:

Click Next to start Restoring as below. Note: Corresponding IDE should not be running.

Note: Restore means copy backup contents and overwrite to original place without clearing. So some settings, such as user-defined environment variables, can be kept after restoring. Our tool only writes old user-defined variables into registry without deleting existing ones. So variables after restoring = variables before restoring + variables in backup.

 

Clean IDE History

This Tool can be used to Clean IDE Opened Projects and Files History.
You can clean the history just by checking them and clicking Next. Notice: Please make sure that IDE are NOT running when executing this Tool.
