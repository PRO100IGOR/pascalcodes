Basic Operation

CnPack Menu

CnPack IDE Wizards is integrated to Delphi/C++Builder IDE. After installing CnWizards, a new menu item named 'CnPack' will appear in IDE main menubar. Click this menu item to show the drop-down menu as below:

Every menu item is corresponding to one wizard in IDE. Our wizards are in development, so you may see more menu items than here.

Some wizards are only enabled in certain circumstance. For examlpe, Component Selector Wizard is only enabled when in Form Designer. If Source Editor is active, Component Selector menu item will be gray.

Languages can be choosen in 'Languages' menu item. Simplified Chinese, Traditional Chinese and English are supported now.

You can see help in 'Help' Meun item. It also includes 'Bug Report & Suggestions', 'Tip of Day' and visiting our website, etc.

Shortcuts in CnPack menu can be customized. If you don't want to see some wizards in the menu, you can set them to NOT active. For more details, please see CnWizards Settings.

 

Toolbar Buttons

Beside menu items and shortcuts, Toolbuttons also can be put on to IDE toolbars. You can:

Right click IDE toolbar, select 'Customize...' in popupmenu.

Or: Select 'View' | 'Toolbars' | 'Customize...' in IDE menu.

A setting dialog appears.

All commands are listed in 'Commands' box. You can drag CnWizards' commands to IDE toolbar.

If you installed Other tools supporting addtional toolbar, such as GExperts. You can put CnWizards' toolbuttons in it, too.

 

Wizards Hint

When you run a wizards first time. A hint window will popuped to let you know its function.

If you don't want to see all hint windows, you can checked 'Disable All'. Or disable it in Environments and Upgrade.
