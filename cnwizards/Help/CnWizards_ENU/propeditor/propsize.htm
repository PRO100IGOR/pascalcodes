Constraints Editor

Using this property editor can help you to edit Constraints property.

  • Original Value: Constraints' original value.

  • New Value: New value to set.

  • Reset: Reset values to zero.

  • Fixed Size: Set current height/width to max/min height/width.

  • Fixed Width: Set current width to max/min width.

  • Fixed Height: Set current height to max/min height

  • Current Value as Max Value: Set current height/width to max height/width.

  • Current Value as Min Value: Set current height/width to min height/width.
