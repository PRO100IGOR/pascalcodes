选择块浮动按钮

该工具提供了一组与选择文本相关功能的快捷操作方式。当用户在代码编辑器中选择了一段文本时，会自动在编辑器左边显示一个浮动按钮，点击该按键下拉出相关的操作菜单。

使用界面如下图所示：

编辑菜单

  • 复制到剪帖板：复制当前选择块。

  • 追加复制到剪贴板：复制当前选择块并追加到剪贴板。

  • 剪切到剪贴板：剪切当前选择块。

  • 追加剪切到剪贴板：剪切当前选择块并追加到剪贴板。

  • 在当前位置重复：将当前选择块原样复制一份在当前位置并保持块选择。

  • 删除：删除选择块。

  • 复制 HTML 到剪帖板：将当前选择块内容以语法高亮的 HTML 格式复制到剪贴板，见 带格式代码输出工具。

  • 保存到文件：将当前选择块内容保存到文件中。

   

大小写转换菜单

  • 转换为小写：将当前选择文本全部转换成小写。

  • 转换为大写：将当前选择文本全部转换成大写。

  • 大小写互换：将当前选择文本大小写互换。


格式处理菜单

  • 代码缩进：使用 IDE 中设置的缩进量缩进当前选择块，可使用 Tab 键快速缩进。

  • 代码缩进指定列：使用指定的缩进量缩进当前选择块。

  • 代码反缩进：使用 IDE 中设置的缩进量反缩进当前选择块，可使用 Shift+Tab 键快速反缩进。

  • 代码反缩进指定列：使用指定的缩进量反缩进当前选择块。

   

代码注释菜单

  • 注释代码块：使用 // 注释当前选择块。见 代码注释工具。

  • 取消代码块注释：取消使用 // 的注释块。见 取消代码注释工具。

  • 反转代码块注释：反转选择块的代码注释。见 反转代码注释工具。

代码嵌入菜单

  • 嵌入到代码中：根据需要将当前选择的代码块缩进并嵌入到指定的语句中。

  • 设置：显示代码嵌入设置对话框如下图所示。

  • 项目列表

    • 增加：增加一项新的内容。

    • 删除：删除当前选择的项目。

    • 上、下：向上或向下移动当前选择的项目。

    • 导入：从文件中导入项目到当前列表，可以选择替换或追加方式。

    • 导出：将当前列表中的项目导出到数据文件。

  • 代码嵌入项

    • 标题：显示在菜单中的标题。

    • 快捷键：定义当前项目的快捷键。

    • 代码块： 如果选择了“行块处理模式”，则在执行代码嵌入时将选择区域首尾扩展到整行进行处理。在行块模式下，可以设置块缩进级数来对代码块进行缩进。

    • 头部：插入到选择块前面的代码。如果是行块模块，可以选择自动缩进和缩进级数。使用 "|" 可以定位光标。

    • 尾部：插入到选择块后面的代码。如果是行块模块，可以选择自动缩进和缩进级数。使用 "|" 可以定位光标。

  • 确定、取消：保存或取消当前的设置。

  • 帮助：显示帮助页面。

组替换菜单

  • 组替换功能：根据设置对当前选择块中的内容执行批量替换。

  • 设置：显示组替换设置对话框如下图所示。

  • 项目列表

    • 增加：增加一项新的内容。

    • 删除：删除当前选择的项目。

    • 上、下：向上或向下移动当前选择的项目。

    • 导入：从文件中导入项目到当前列表，可以选择替换或追加方式。

    • 导出：将当前列表中的项目导出到数据文件。

  • 组替换项

    • 标题：显示在菜单中的标题。

    • 快捷键：定义当前项目的快捷键。

    • 替换项：列出需要执行替换的内容。

    • 增加：增加一项新的内容。

    • 删除：删除当前选择的项目。

    • 上、下：向上或向下移动当前选择的项目。

    • 源文本：需要进行替换的源字符串。

    • 替换为：替换后的目标字符串。

    • 忽略大小写：替换当前项时是否忽略大小写。

    • 整词匹配：替换时是否要求匹配的文本为独立的单词。

  • 确定、取消：保存或取消当前的设置。

  • 帮助：显示帮助页面。

Web 搜索菜单

  • Web 搜索功能：打开搜索链接页面，查找当前选择的内容。

  • 设置：显示 Web 搜索设置对话框如下图所示。

  • 项目列表

    • 增加：增加一项新的内容。

    • 删除：删除当前选择的项目。

    • 上、下：向上或向下移动当前选择的项目。

    • 导入：从文件中导入项目到当前列表，可以选择替换或追加方式。

    • 导出：将当前列表中的项目导出到数据文件。

  • Web 搜索项

    • 标题：显示在菜单中的标题。

    • 快捷键：定义当前项目的快捷键。

    • 搜索链接：指定进行搜索的可执行文件或网页链接，%s 表示当前选择内容。

  • 确定、取消：保存或取消当前的设置。

  • 帮助：显示帮助页面。

　

其它菜单

  • 赋值交换：将当前选择的源代码中的赋值代码左右交换，即将 A := B 的代码变换为 B := A。见 赋值交换工具。

  • 转换为字符串：将当前选择的源代码转换成多行字符串常量，这样可以省去输入引号和换行字符的麻烦。见 转换为字符串工具。

  • 插入颜色：打开颜色对话框并用选择的颜色值替换当前选择的文本。见 插入颜色工具。

  • 插入日期时间：打开插入日期对话框并用日期时间值替换当前选择的文本。见 插入日期时间工具。

  • 转换为字符串：将当前选择的代码行按字母排序。见 选择行排序工具。

　

工具设置

其设置界面如下图所示：

  • 块选择时显示浮动工具按钮：是否启动选择块浮动工具按钮功能。

  • 在编辑器右键菜单中显示浮动工具按钮下拉菜单：是否在编辑器右键菜单中插入一项子菜单，其内容是浮动工具按钮下拉菜单中的菜单项，供右键菜单调用。
