函数过程列表

该功能可显示源码文件中的所有函数、过程与方法名，并支持匹配查找快速定位，Pascal 源文件还支持类名、属性和接口等。默认显示对象为当前源文件。用户可选择工程/工程组中的其他单一文件或所有文件、以及所有打开文件等。

另外，该专家将在 IDE 的编辑器中加入一工具栏，实时显示在编辑的 Pascal 源文件光标处的类型声明以及函数过程声明等，可通过下拉的方式快速定位。

  • 跳转至所选过程：在 IDE 中跳转到所选择的过程实现代码处。

  • 复制函数或过程名：复制所选择的函数或过程名称到剪贴板。

  • 按函数/过程名开头匹配：查找的时候，输入的字符从函数/过程名的头部开始匹配，函数/过程名头部含有输入字符的才能显示。

  • 按函数/过程名的所有位置匹配：查找的时候，输入的字符匹配函数/过程名的任何部分，函数/过程名含有输入字符的才能显示。

  • 在编辑器中显示函数列表工具栏：是否允许在 IDE 的代码编辑器中显示函数列表工具栏。此工具栏能实时显示在编辑的 Pascal/C/C++ 源文件光标处的类名或函数过程名，并能通过下拉的方式快速定位。同时还有按钮允许快速跳至当前 Pascal 单元的 interface 和 implementation 处。

  • 字体：更改列表框字体。

  • 帮助：显示帮助信息。

  • 退出：关闭此窗口。

  • 隐藏/显示预览窗口：隐藏或显示选中函数/过程实现部分的预览栏。当所选择的过程函数所在的文件未在 IDE 中打开时，预览不可用。
