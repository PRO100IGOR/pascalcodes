IDE 专家基本操作

专家主菜单

IDE 专家包是紧密集成在 Delphi/C++Builder IDE 环境中的，用户安装专家包后，会在 IDE 主菜单上看到一个新的菜单项 CnPack，点击该菜单项，显示菜单列表如下图所示：

其中，每一个菜单项对应着一个独立的 IDE 专家，专家包当前仍在持续开发中，您看的菜单列表内容可能比上图要多。

某些专家只有在特定条件下才可使用，如“组件选取”专家要求当前活跃窗口为窗体设计器，如果用户正在编辑源代码，则该选项显示变灰。

“Languages”子菜单下可以选择专家界面语言，当前支持 简体中文、繁体中文、英文 三种语言。

“帮助”子菜单下可以查看专家帮助、报告错误、访问开发网站、显示版权信息和每日一帖等。

菜单上的所有快捷键，用户都可以自己定制，如果希望禁用某些专家让其不显示在菜单中，可以在设置界面中设置，详见 专家包设置。

　

工具栏按钮

除了使用菜单项和快捷键调用外，您还可以将专家工具放置到 IDE 的工具栏中，设置方法如下：

方法一：右键点击 IDE 工具栏，在弹出菜单中选择“Customize...”。

方法二：选择 IDE 主菜单下的“View”|“Toolbars”|“Customize...”。

弹出如下设置窗口：

“Commands”页中列出了所有可用的命令，您可以将 CnPack IDE 工具直接拖到 IDE 工具栏上。

下图显示了一个定制的 IDE 工具栏：

有的专家是带子菜单项的，如果您将带子菜单项的工具放到工具栏中，点击它会显示弹出一个下拉菜单，内容与其子菜单项相同。

如下图所示：

如果您安装了 GExperts 等支持附加工具栏的其它第三方工具，同样可以在里面使用。

　

专家提示信息

为了帮助您快速掌握专家功能，在您第一次使用专家工具时，您可以看到类似这样的提示功能：

如果您不需要显示所有专家的提示信息，您可以勾选“关闭所有提示”复选框或在 环境及升级设置 中关闭该功能。

 

每日一帖

如果您不需要显示“每日一帖”，可不勾选“总是在启动时显示本页”。
