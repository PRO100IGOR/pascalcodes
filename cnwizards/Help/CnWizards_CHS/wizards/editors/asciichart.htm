ASCII 字符表

该工具能够显示一个 ASCII 字符表，供用户在开发过程中参考和输入。该窗体可停靠在 IDE 中。以下是其界面。

  • ASCII 显示区：显示了128个 ASCII 字符，左键单击某一个字符可以在下面的字符转换结果编辑栏中输入它，右键单击某一个字符时会在代码编辑器的当前光标处输入它，如果该字符属于可显示的 ASCII 字符则输入该字符，否则以 # 加数字的形式输入。

  • 字体与字号调整：窗体上部的下拉框和字号调整框可以用来调整显示的 ASCII 字符的字体。

  • $ 按钮：对可用来切换 ASCII 值以十进制显示还是十六进制显示，也控制着下面字符转换的格式。

  • > 按钮：可以在标准 ASCII 字符区和扩展 ASCII 字符区切换。

  • 放大图：可以显示当前字符轮廓的三倍放大图。

  • 字符转换栏：在上方的字符编辑输入栏输入字符后按转换按钮，即可在下方字符转换结果编辑栏中得到转换结果，其中，不可视字符或扩展 ASCII 码字符将被转换成以 # 加数字的形式。

  • 状态栏：能够显示当前选中的 ASCII 字符的十进制码、十六进制码和该字符本身。
