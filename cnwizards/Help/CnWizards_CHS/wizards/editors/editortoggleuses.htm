切换引用区

此工具可供光标在当前位置和 uses/#include 之间跳转，用于快速引用其他单元。默认快捷键是 Ctrl+Alt+U。对于 Pascal 文件，执行该工具可让光标从当前位置跳至光标上部最近的 uses 部分，如第一次跳至 implementation 处的 uses 时，连续快速按两次热键能跳至 interface 处的 uses。对于 C 或 H 文件，可跳至头部 #include 处。输入完毕后再按一次快捷键或 ESC 返回。
