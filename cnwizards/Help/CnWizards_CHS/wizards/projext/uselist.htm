引用单元列表

本功能是 IDE 中引用单元功能的扩展。本功能可显示当前工程中供引用的所有单元名、类型、所属工程名、大小和文件类型等，可根据文件名查找过滤，其中红色为已打开项目。

  • 引用所选单元：当前单元引用所选择的单元，可以是多个单元。

  • 显示所选单元属性：显示所选择的单元文件的属性对话框。

  • 复制单元名：复制所选择的单元名称到剪贴板。

  • 选择所有单元：选中单元列表中的所有单元。

  • 取消选择：取消单元列表中的选择。

  • 反选单元：反向选择单元列表中的已选择单元。

  • 按单元名开头匹配：查找的时候，输入的字符从单元名的头部开始匹配，单元名头部含有输入字符的才能显示。

  • 按单元名的所有位置匹配：查找的时候，输入的字符匹配单元名的任何部分，单元名含有输入字符的才能显示。

  • 挂接 IDE：挂接引用单元列表到 IDE，当打开 File 菜单中的“Use Unit...”菜单项时，将会打开此引用单元列表。另外由于 IDE 本身的特性，此挂接选项还会同时控制插入 Frame 对话框的挂接。此功能参见“Frame 列表”。

  • 字体：更改列表框字体。

  • 帮助：显示此帮助。

  • 关闭：关闭此窗口。
