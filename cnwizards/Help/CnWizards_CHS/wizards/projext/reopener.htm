打开历史文件

该功能可分类记录曾经打开的文件，为再次打开以前的文件提供了方便。默认的热键是“Shift+Ctrl+O”，单击此菜单后，弹出的界面如图所示：

　　窗口左边是文件分类，包括工程组文件、工程文件、源码文件（Delphi 下是 pas 文件，C++Builder 下是 cpp 和 c 等文件）、包文件、其他文件等，还包括一收藏夹。用户可将常打开的文件拖动到收藏夹中供日后频繁打开。

　　窗口右边是文件列表，记录曾经打开过的文件的文件名、路径、最后打开和关闭时间等。IDE 中打开的文件会在此专家中自动记录并分类列出。窗口顶部是工具栏。

  • 打开：在 IDE 中打开所选择的文件，可以是多个。如果是单独的源文件，则此窗口不会关闭，可供连续打开多个文件使用。如选择的是工程或工程组，打开后，本窗口将关闭。双击选中的文件能完成同样的功能。

  • 删除：从列表中删除所选中的文件，文件本身不会从磁盘上删除。

  • 选项：弹出选项设置对话框，对打开历史文件专家进行设置。

  • 收藏：将选中的文件移至收藏夹，相当于拖动文件到左边的收藏夹一栏。

  • 帮助：显示帮助信息。

  • 退出：关闭本窗口。

　　右键单击选中的文件，会弹出上下文菜单，包括以下几项：

  • 打开文件：等同于工具栏的“打开”命令，在 IDE 中打开所选文件。

  • 复制文件名到剪贴板：将选中文件的文件名复制到剪贴板，不包括路径。

  • 复制文件路径到剪贴板：将选中文件的所在路径复制到剪贴板，不包括文件名。

  • 打开所在的文件夹：在资源管理器中打开所选文件所在的文件夹。

　

选项设置

单击“选项”按钮，将弹出下图所示的对话框：

  • 普通选项：

    • 忽略 Unit1, Project1：设置是否记录 Unit1 和 Project1 这样缺省的文件名和工程名。

    • 默认打开页：设置本专家显示时的默认页，用户可选择最需要的页面如源码文件页等。

    • 保留标题大小与排序方法：勾选此项后，各个文件列表框中的按哪些栏排序以及各栏调整后的宽度会被保存，下次打开窗口时可回复到调整后的宽度与排序栏。

    • 本地日期格式：勾选此项后，文件日期将按本地格式记录。

  • 文件数设置：可设置各类文件列表的最大容纳文件数。

  • 确定：关闭此对话框，保存选项设置并生效。

  • 取消：关闭此对话框，不保存选项。
