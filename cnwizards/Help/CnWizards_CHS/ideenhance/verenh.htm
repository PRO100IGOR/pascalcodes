版本信息扩展专家

版本信息扩展专家能够在编译时自动升级工程的版本号，并且能够将编译时间插入版本信息中。该专家的功能仅在特定版本的 IDE 中有效。当前不支持 Delphi 5 和 C++Builder 5/6。

该专家的设置窗口如下图所示：

  • 版本信息中插入编译时刻：在版本信息中插入最后一次编译的时刻。此功能依赖于工程选项的版本信息列表中是否支持编译时间（LastCompiledTime）这一项，因而可能在部分 IDE 下无效。

  • 编译时自动增加 Build 号：在每一次编译时（而不仅仅是 Build 时）将 Build 版本号加一。

注：以上两选项只在工程选项中包含版本信息时有效，选项页中“Include version information in project”未选中时无效。
