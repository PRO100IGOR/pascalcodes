属性编辑器介绍

CnPack IDE 专家包中包含了一些属性编辑器，以扩展 IDE 自身对象查看器的功能。

这些属性编辑器可以在 属性、组件编辑器设置 中进行配置。

专家包的属性编辑器支持动态装卸，设置修改后立即生效。

　

布尔属性编辑器

该编辑器可以在布尔属性中显示一个检查框，以直观地显示其状态。

　

集合属性编辑器

IDE 默认的集合编辑器只允许用扩展方式编辑子项，该编辑器允许用户使用下拉列表的方式进行编辑，同时还支持直接输入集合值。

　

字符串、标题属性编辑器

该编辑器允许在 Caption 及其它 String 属性中输入多行文本。

参见 多行文本编辑器。

　

Hint 属性编辑器

该编辑器允许在 Hint 属性中输入多行文本，支持长格式和短格式的 Hint。

参见 多行文本编辑器。

　

字符串列表属性编辑器

该编辑器扩展了 IDE 默认的字符串列表属性编辑器，支持查找、保存、装载等操作。

参见 多行文本编辑器。

　

文件名属性编辑器

该编辑器允许用文件对话框来选择 FileName 属性的值。

　

Constraints属性编辑器

该编辑器可以方便地修改控件的 Constraints 属性。

参见 Constraints 编辑器。

　

字体属性编辑器

该编辑器可以在对象查看器中显示字体的详细信息来取代默认的 (TFont) 显示。

　

控件滚动条属性编辑器

该编辑器可以显示滚动条的详细信息，包括是否可见，以及位置等。

　

Align 属性编辑器

该编辑器可以为 Align 属性显示一个直观的图形。

　

组件名称属性编辑器

该编辑器调用 组件前缀专家 中的功能来为组件规范化重命名。
