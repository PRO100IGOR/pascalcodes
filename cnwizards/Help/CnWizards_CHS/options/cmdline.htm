命令行参数

专家包支持在 IDE 的命令行中使用以下参数：

  • 不启动专家包：如果在命令行中使用 -nocn 参数，当 IDE 启动时 CnWizards 将不创建任何专家实例。

  • 显示专家启动窗口：如果在命令行中使用 -swcn 参数，当 IDE 启动时会弹出一个窗口，允许用户选择需要创建的专家。这个功能通常用于调试，在 IDE 启动时按住 Shift 键也可以达到同样的目的。

  • 指定专家使用的注册表键：如果在命令行中使用 -cnregXXXX 参数，专家包所有的注册表设置将会放在 HKEY_CURRENT_USER\Software\XXXX 下面。专家包默认所有 IDE 使用相同的参数设置，使用这个功能可以为相同或不同的 IDE 指定不同的参数设置。
