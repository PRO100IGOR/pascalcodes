選擇塊浮動按鈕

該工具提供了一組與選擇文本相關功能的快捷操作方式。當用戶在代碼編輯器中選擇了一段文本時，會自動在編輯器左邊顯示一個浮動按鈕，點擊該按鍵下拉出相關的操作功能表。

使用介面如下圖所示：

編輯功能表

  • 複製到剪帖板：複製當前選擇塊。

  • 追加複製到剪貼板：複製當前選擇塊並追加到剪貼板。

  • 剪切到剪貼板：剪切當前選擇塊。

  • 追加剪切到剪貼板：剪切當前選擇塊並追加到剪貼板。

  • 在當前位置重複：將當前選擇塊原樣複製一份在當前位置並保持塊選擇。

  • 刪除：刪除選擇塊。

  • 複製 HTML 到剪帖板：將當前選擇塊內容以語法高亮的 HTML 格式複製到剪貼板，見 帶格式代碼輸出工具。

  • 保存到檔：將當前選擇塊內容保存到檔中。

   

大小寫轉換菜單

  • 轉換為小寫：將當前選擇文本全部轉換成小寫。

  • 轉換為大寫：將當前選擇文本全部轉換成大寫。

  • 大小寫互換：將當前選擇文本大小寫互換。


格式處理功能表

  • 代碼縮進：使用 IDE 中設置的縮進量縮進當前選擇塊，可使用 Tab 鍵快速縮進。

  • 代碼縮進指定列：使用指定的縮進量縮進當前選擇塊。

  • 代碼反縮進：使用 IDE 中設置的縮進量反縮進當前選擇塊，可使用 Shift+Tab 鍵快速反縮進。

  • 代碼反縮進指定列：使用指定的縮進量反縮進當前選擇塊。

   

代碼注釋功能表

  • 注釋代碼塊：使用 // 注釋當前選擇塊。見 代碼注釋工具。

  • 取消代碼塊注釋：取消使用 // 的注釋塊。見 取消代碼注釋工具。

  • 反轉代碼塊注釋：反轉選擇塊的代碼注釋。見 反轉代碼注釋工具。

代碼嵌入功能表

  • 嵌入到代碼中：根據需要將當前選擇的代碼塊縮進並嵌入到指定的語句中。

  • 設置：顯示代碼嵌入設置對話方塊如下圖所示。

  • 項目列表

    • 增加：增加一項新的內容。

    • 刪除：刪除當前選擇的項目。

    • 上、下：向上或向下移動當前選擇的項目。

    • 導入：從檔中導入專案到當前列表，可以選擇替換或追加方式。

    • 導出：將當前列表中的項目導出到資料檔案。

  • 代碼嵌入項

    • 標題：顯示在功能表中的標題。

    • 快捷鍵：定義當前項目的快捷鍵。

    • 代碼塊： 如果選擇了“行塊處理模式”，則在執行代碼嵌入時將選擇區域首尾擴展到整行進行處理。在行塊模式下，可以設置塊縮進級數來對代碼塊進行縮進。

    • 頭部：插入到選擇塊前面的代碼。如果是行塊模組，可以選擇自動縮進和縮進級數。使用 "|" 可以定位游標。

    • 尾部：插入到選擇塊後面的代碼。如果是行塊模組，可以選擇自動縮進和縮進級數。使用 "|" 可以定位游標。

  • 確定、取消：保存或取消當前的設置。

  • 說明：顯示說明頁面。

組替換菜單

  • 組替換功能：根據設置對當前選擇塊中的內容執行批量替換。

  • 設置：顯示組替換設置對話方塊如下圖所示。

  • 項目列表

    • 增加：增加一項新的內容。

    • 刪除：刪除當前選擇的項目。

    • 上、下：向上或向下移動當前選擇的項目。

    • 導入：從檔中導入專案到當前列表，可以選擇替換或追加方式。

    • 導出：將當前列表中的項目導出到資料檔案。

  • 組替換項

    • 標題：顯示在功能表中的標題。

    • 快捷鍵：定義當前項目的快捷鍵。

    • 替換項：列出需要執行替換的內容。

    • 增加：增加一項新的內容。

    • 刪除：刪除當前選擇的項目。

    • 上、下：向上或向下移動當前選擇的項目。

    • 源文本：需要進行替換的源字串。

    • 替換為：替換後的目標字串。

    • 忽略大小寫：替換當前項時是否忽略大小寫。

    • 整詞匹配：替換時是否要求匹配的文本為獨立的單詞。

  • 確定、取消：保存或取消當前的設置。

  • 說明：顯示說明頁面。

Web 搜索菜單

  • Web 搜索功能：打開搜索鏈結頁面，查找當前選擇的內容。

  • 設置：顯示 Web 搜索設置對話方塊如下圖所示。

  • 項目列表

    • 增加：增加一項新的內容。

    • 刪除：刪除當前選擇的項目。

    • 上、下：向上或向下移動當前選擇的項目。

    • 導入：從檔中導入專案到當前列表，可以選擇替換或追加方式。

    • 導出：將當前列表中的項目導出到資料檔案。

  • Web 搜索項

    • 標題：顯示在功能表中的標題。

    • 快捷鍵：定義當前項目的快捷鍵。

    • 搜索鏈結：指定進行搜索的可執行檔或網頁鏈結，%s 表示當前選擇內容。

  • 確定、取消：保存或取消當前的設置。

  • 說明：顯示說明頁面。

　

其他菜單

  • 賦值交換：將當前選擇的源代碼中的賦值代碼左右交換，即將 A := B 的代碼變換為 B := A。見 賦值交換工具。

  • 轉換為字串：將當前選擇的源代碼轉換成多行字串常量，這樣可以省去輸入引號和換行字元的麻煩。見 轉換為字串工具。

  • 插入顏色：打開顏色對話方塊並用選擇的顏色值替換當前選擇的文本。見 插入顏色工具。

  • 插入日期時間：打開插入日期對話框並用日期時間值替換當前選擇的文本。見 插入日期時間工具。

  • 轉換為字串：將當前選擇的代碼行按字母排序。見 選擇行排序工具。

　

工具設置

其設置介面如下圖所示：

  • 塊選擇時顯示浮動工具按鈕：是否啟動選擇塊浮動工具按鈕功能。

  • 在編輯器右鍵菜單中顯示浮動工具按鈕下拉菜單：是否在編輯器右鍵菜單中插入一項子菜單，其內容是浮動工具按鈕下拉菜單中的菜單項，供右鍵菜單調用。
