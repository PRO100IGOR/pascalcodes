前進後退擴展

IDE 本身提供的前進、後退功能只對識別字聲明跳轉有效，該擴展通過跟蹤當前行號的改變來提供更方便的前進、後退功能。當編輯器游標移動超過指定行（默認是 5行）時，工具會自動記住移動前的位置，並允許使用“後退”功能退回。

比如在編寫代碼時，需要到 uses 區添加一個新的引用單元，可以直接用捲軸找到 uses 位置並定位游標，輸入完單元名稱後再用“後退”功能，即可回到原來編輯的位置。

該擴展取代了 IDE 原來的按鈕，後退功能的下拉功能表如下：

  • IDE 後退：調用 IDE 自己的後退功能。在點擊後退按鈕時按住“Shift”、“Alt”或“Ctrl”中任意一鍵，都會直接調用 IDE 的後退。

  • IDE 後退列表：顯示 IDE 自己的後退歷史列表。

  • 後退列表項：後退歷史堆疊中的列表，標題為檔案名和行號，點擊可直接跳轉到該位置。

  • 暫用 IDE 跳轉功能：在點擊按鈕時暫時使用 IDE 自己的跳轉功能。該狀態退出後不保存，如果需要關閉前進後退擴展，請在設置中禁用。

  • 編輯器擴展配置：打開 代碼編輯器擴展設置 視窗，設置內容見後。

　

該擴展功能的設置介面如下圖所示：

  • 擴展代碼編輯器的前進、後退按鈕功能：是否啟動前進後退擴展。

  • 新記錄的最小行間隔：當游標移動到與上一處位置間隔指定的行數時，產生新的後退記錄。

  • 最大列表項數：列表中保存的歷史記錄最大值。
