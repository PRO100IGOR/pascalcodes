Repository 專家

CnPack IDE Wizards 自帶了一些 Repository 專家，也就是用來生成特定的新專案或單元的嚮導。可以在 IDE 的 File 菜單下的 New...對話方塊中的 CnPack 頁中訪問到它們。Repository 專家也在不斷的開發中，您看到的嚮導可能比現在要多。

另外，在 CnPack 功能表中的 “Repository 列表”一項中，也可以通過點擊其子功能表來啟動各個 CnPack 專家包自帶的 Repository 專家。
