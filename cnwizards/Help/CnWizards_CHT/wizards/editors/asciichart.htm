ASCII 字元表

該工具能夠顯示一個 ASCII 字元表，供用戶在開發過程中參考和輸入。該表單可停靠在 IDE 中。以下是其介面。

  • ASCII 顯示區：顯示了128個 ASCII 字元，左鍵單擊某一個字元可以在下面的字元轉換結果編輯欄中輸入它，右鍵單擊某一個字元時會在代碼編輯器的當前游標處輸入它，如果該字元屬於可顯示的 ASCII 字元則輸入該字元，否則以 # 加數位的形式輸入。

  • 字體與字型大小調整：表單上部的下拉清單和字型大小調整框可以用來調整顯示的 ASCII 字元的字體。

  • $ 按鈕：對可用來切換 ASCII 值以十進位顯示還是十六進位顯示，也控制著下面字元轉換的格式。

  • > 按鈕：可以在標準 ASCII 字元區和擴展 ASCII 字元區切換。

  • 放大圖：可以顯示當前字元輪廓的三倍放大圖。

  • 字元轉換欄：在上方的字元編輯輸入欄輸入字元後按轉換按鈕，即可在下方字元轉換結果編輯欄中得到轉換結果，其中，不可視字元或擴展 ASCII 碼字元將被轉換成以 # 加數位的形式。

  • 狀態欄：能夠顯示當前選中的 ASCII 字元的十進位碼、十六進位碼和該字元本身。
