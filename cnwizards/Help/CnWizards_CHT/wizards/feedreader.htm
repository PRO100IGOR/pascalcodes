源閱讀專家

該專家用來訂閱 RSS 和 ATOM 源，並將內容顯示在 IDE 的狀態列上面。

源內容顯示

源標題將動態顯示在 IDE 下方的狀態列上並顯示為超連結，點擊可打開相關頁面。

滑鼠移入狀態列後，在右方會顯示三個按鈕，分別是：上一個、設置、下一個。

右鍵點擊可顯示功能表。

  • 上一條：切換到上一條內容。

  • 下一條：切換到下一條內容。

  • 全部更新：更新當前訂閱的所有頻道。

  • 關閉：關閉專家。

  • 設置：打開設置對話方塊。

　

源閱讀專家設置

設置對話方塊如圖所示：

　

  • 訂閱 CnPack 頻道：是否訂閱 CnPack 官方頻道，建議啟用。

  • 訂閱 CnPack 合作夥伴頻道：是否訂閱 CnPack 合作夥伴官方頻道，建議啟用。

  • 隨機：是否隨機顯示源內容。

  • 切換週期：自動切換源條目的週期。

  • 源列表：自訂的源列表，支援 RSS 和 ATOM 格式。

    • 增加：增加一個新的源。

    • 刪除：刪除當前選擇的源。

    • 標題：標識當前源，只用於顯示。

    • 連結：源連結，支援 RSS 和 ATOM 格式。

    • 更新週期：設置自動更新源內容的時間：

    • 限制：限制只顯示最近的條目數量，0為不限制。

  • 確定：保存修改，關閉對話方塊。

  • 取消：放棄修改，關閉對話方塊。

  • 幫助：顯示本說明。
