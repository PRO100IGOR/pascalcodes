引用單元列表

本功能是 IDE 中引用單元功能的擴展。本功能可顯示當前工程中供引用的所有單元名、類型、所屬工程名、大小和文件類型等，可根據文件名查找過濾，其中紅色為已打開項目。

  • 引用所選單元：當前單元引用所選擇的單元，可以是多個單元。

  • 顯示所選單元屬性：顯示所選擇的單元文件的屬性對話框。

  • 複製單元名：複製所選擇的單元名稱到剪貼板。

  • 選擇所有單元：選中單元列表中的所有單元。

  • 取消選擇：取消單元列表中的選擇。

  • 反選單元：反向選擇單元列表中的已選擇單元。

  • 按單元名開頭匹配：查找的時候，輸入的字符從單元名的頭部開始匹配，單元名頭部含有輸入字符的才能顯示。

  • 按單元名的所有位置匹配：查找的時候，輸入的字符匹配單元名的任何部分，單元名含有輸入字符的才能顯示。

  • 掛接 IDE：掛接引用單元列表到 IDE，當打開 File 菜單中的「Use Unit...」菜單項時，將會打開此引用單元列表。另外由於 IDE 本身的特性，此掛接選項還會同時控制插入 Frame 對話框的掛接。此功能參見「Frame 列表」。

  • 字體：更改列表框字體。

  • 幫助：顯示此幫助。

  • 關閉：關閉此窗口。
