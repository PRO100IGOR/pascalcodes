已引用單元列表

顯示當前單元所引用的單元名列表，可根據文件名查找過濾，可在搜索路徑中搜索選中文件並打開。本功能支持 Pascal 和 C/C++ 源文件。對於 Pascal 源文件，本功能能列出它 uses 的文件名列表並顯示是在 interface 部分還是 implementation 部分；對於 C/C++ 源文件，本功能可列出它 include 的文件名列表。

  • 打開所選單元：在 IDE 中打開所選擇的單元，可以是多個單元。

  • 複製單元名：複製所選擇的單元名稱到剪貼板。

  • 選擇所有單元：選中單元列表中的所有單元。

  • 取消選擇：取消單元列表中的選擇。

  • 反選單元：反向選擇單元列表中的已選擇單元。

  • 按單元名開頭匹配：查找的時候，輸入的字符從單元名的頭部開始匹配，單元名頭部含有輸入字符的才能顯示。

  • 按單元名的所有位置匹配：查找的時候，輸入的字符匹配單元名的任何部分，單元名含有輸入字符的才能顯示。

  • 打開多個單元提示：打開多個單元時提示。

  • 字體：更改列表框字體。

  • 幫助：顯示此幫助。

  • 關閉：關閉此窗口。
