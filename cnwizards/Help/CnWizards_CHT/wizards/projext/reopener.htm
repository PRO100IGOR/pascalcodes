打開歷史文件

該功能可分類記錄曾經打開的文件，為再次打開以前的文件提供了方便。默認的熱鍵是「Shift+Ctrl+O」，單擊此菜單後，彈出的界面如圖所示：

　　窗口左邊是文件分類，包括工程組文件、工程文件、源碼文件（Delphi 下是 pas 文件，C++Builder 下是 cpp 和 c 等文件）、包文件、其他文件等，還包括一收藏夾。用戶可將常打開的文件拖動到收藏夾中供日後頻繁打開。

　　窗口右邊是文件列表，記錄曾經打開過的文件的文件名、路徑、最後打開和關閉時間等。IDE 中打開的文件會在此專家中自動記錄並分類列出。窗口頂部是工具欄。

  • 打開：在 IDE 中打開所選擇的文件，可以是多個。如果是單獨的源文件，則此窗口不會關閉，可供連續打開多個文件使用。如選擇的是工程或工程組，打開後，本窗口將關閉。雙擊選中的文件能完成同樣的功能。

  • 刪除：從列表中刪除所選中的文件，文件本身不會從磁盤上刪除。

  • 選項：彈出選項設置對話框，對打開歷史文件專家進行設置。

  • 收藏：將選中的文件移至收藏夾，相當於拖動文件到左邊的收藏夾一欄。

  • 幫助：顯示幫助信息。

  • 退出：關閉本窗口。

　　右鍵單擊選中的文件，會彈出上下文菜單，包括以下幾項：

  • 打開文件：等同於工具欄的「打開」命令，在 IDE 中打開所選文件。

  • 複製文件名到剪貼板：將選中文件的文件名複製到剪貼板，不包括路徑。

  • 複製文件路徑到剪貼板：將選中文件的所在路徑複製到剪貼板，不包括文件名。

  • 打開所在的文件夾：在資源管理器中打開所選文件所在的文件夾。

　

選項設置

單擊「選項」按鈕，將彈出下圖所示的對話框：

  • 普通選項：

    • 忽略 Unit1, Project1：設置是否記錄 Unit1 和 Project1 這樣缺省的文件名和工程名。

    • 默認打開頁：設置本專家顯示時的默認頁，用戶可選擇最需要的頁面如源碼文件頁等。

    • 保留標題大小與排序方法：勾選此項後，各個文件列表框中的按哪些欄排序以及各欄調整後的寬度會被保存，下次打開窗口時可回復到調整後的寬度與排序欄。

    • 本地日期格式：勾選此項後，文件日期將按本地格式記錄。

  • 文件數設置：可設置各類文件列表的最大容納文件數。

  • 確定：關閉此對話框，保存選項設置並生效。

  • 取消：關閉此對話框，不保存選項。
