Frame 列表

本功能是 IDE 中插入 Frame 對話框功能的擴展。本功能可顯示當前工程中可供插入的所有 Frame 的名稱、所屬工程名、大小和文件類型等，可根據文件名查找過濾，其中紅色為已打開項目。

本功能在當點擊組件面板上的 Frame 圖標並在當前窗體上點擊時被調用，取代了 IDE 本身的插入 Frame 對話框。由於 IDE 本身的特性，本取代功能是否使能由「引用單元列表」對話框中的掛接按鈕統一控制，使用時請注意。

  • 將所選 Frame 加入窗體：將所選的 Frame 加入當前窗體。

  • 顯示所選 Frame 的文件屬性：顯示所選擇的 Frame 所在的單元文件的屬性對話框。

  • 複製 Frame 名稱：複製所選擇的 Frame 名稱到剪貼板。

  • 按Frame 名稱開頭匹配：查找的時候，輸入的字符從 Frame 名稱的頭部開始匹配，單元名頭部含有輸入字符的才能顯示。

  • 按 Frame 名稱的所有位置匹配：查找的時候，輸入的字符匹配 Frame 名稱的任何部分，單元名含有輸入字符的才能顯示。

  • 字體：更改列表框字體。

  • 幫助：顯示此幫助。

  • 關閉：關閉此窗口。
