IDE 專家管理工具

該專家專家管理工具通過註冊表進行 IDE 的外部專家管理，支持 Delphi/C++Builder/BDS。

Borland/CodeGear Delphi 系列 IDE 的外部專家註冊信息均保存在註冊表中，本工具能以可視化的方式提供對註冊表中專家註冊信息的管理，包括啟用、禁用等。另外本專家只支持 Win32 類型的 DLL 專家，不支持 BPL 以及 .NET 類型的專家。

該工具主界面如下圖所示：左面列出了支持的 IDE 版本類型，未安裝的 IDE 版本以灰色顯示。當選中一 IDE 類型時，右面會出現其已安裝的 DLL 專家列表，用戶可通過點擊列表框中的復選框方便地啟用、禁用它們。

  • 重新載入：從註冊表中載入選中 IDE 類型的專家信息，界面上的改動將被忽略。

  • 全部保存：保存全部 IDE 中的已改動的專家註冊信息。

  • 新增專家：打開一專家文件並註冊到當前 IDE 中只支持 Win32 DLL 類型的專家文件。

  • 刪除專家：從註冊表中徹底刪除此專家的註冊信息，專家文件本身不受影響。

  • 顯示專家文件屬性：顯示資源管理器中的該專家的文件屬性對話框。

  • 打開所在目錄：在資源管理器中打開該專家文件所在目錄並定位到此專家文件。

  • 外殼關聯菜單：顯示資源管理器中此專家文件的外殼關聯右鍵菜單。

  • 全選：全部啟用當前 IDE 中的專家。

  • 全不選：全部禁用當前 IDE 中的專家。

  • 反向選擇：反向選擇當前 IDE 專家列表中的專家。

  • 幫助：顯示此幫助內容。

  • 關閉：關閉程序退出。如果有 IDE 改動過，則提示保存。
