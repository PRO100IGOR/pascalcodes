ImageList 組件編輯器

該編輯器為 ImageList 提供了強大的設計支援，包括 XP 風格帶 Alpha 通道的圖像清單支援、png 檔支援（自動轉換為 Bitmap）和線上搜索圖示等功能。

下圖為 ImageList 元件編輯器介面：

選中的圖像

  • 圖像預覽：預覽當前選中的圖像，背景棋盤格表示圖像的透明部分。

  • 透明色：如果選中的圖像是通過檔或搜索添加的 png 和 bmp 檔，可以選擇其透明色。點擊下拉清單右邊的按鈕，可從圖像預覽區點取透明色。

  • 使用帶透明通道的 XP 風格圖示：選中該選項後，圖像清單將使用 32 bit 的格式。該選項需要 IDE 本身支援 XPManifest 才會啟用。

  • 選項：如果選中的圖像是通過檔或搜索添加的 png 和 bmp 檔，設定顯示的縮放方式。

圖像清單

  • 增加：從檔案系統中選擇一個或多個影像檔，添加到列表中。支援 bmp、ico 和 png 三種格式。

  • 替換：從檔案系統中選擇一個或多個影像檔，替換掉當前選中的圖像。

  • 刪除：刪除當前選中的圖像。

  • 清空：清空當前圖像清單。

  • 匯出：匯出當前選中的圖像或全部圖像為影像檔，支援 bmp 和 png 格式匯出。

  • 拖動排序：在圖像清單中，可通過選中圖示拖動來進行排序。

  • 圖像大小：可以快速修改圖像清單的大小。如果當前圖像清單不為空，修改大小後會自動對圖像進行縮放。

線上搜索

  • 關鍵字：輸入要搜索的關鍵字，多個關鍵字用空格隔開，目前只支持英文單詞。

  • 搜索：開始搜索圖示。

  • 圖示來源：選擇提供圖示搜索服務的網站。

  • 轉到：在流覽器中打開當前選擇的圖示搜索服務網站。

  • 只包含允許商業授權的圖示：搜索圖示時指定是否在結果中只包含允許在商務軟體中使用的圖示。

  • 增加：將當前選中的圖像添加到左邊的圖像清單中。

  • 替換：用當前選中的圖像替換左邊的圖像清單中選中的圖像。

  • 翻頁：在搜索結果中翻頁。

  • 刷新菜單：刷新當前頁面的顯示。

  • 轉到頁數：跳轉到指定頁數顯示。

  • 在流覽器中打開圖像：直接從網站上訪問當前選中圖像。

  • 搜索圖示集：搜索當前選中圖像所從屬的圖示集，該功能只對部分圖示來源有效。
