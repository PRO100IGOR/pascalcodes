屬性編輯器

屬性編輯器介紹

CnPack IDE 專家包中包含了一些屬性編輯器，以擴展 IDE 自身物件查看器的功能。

這些屬性編輯器可以在 屬性、元件編輯器設置 中進行配置。

專家包的屬性編輯器支援動態裝卸，設置修改後立即生效。

　

布林屬性編輯器

該編輯器可以在布林屬性中顯示一個檢查框，以直觀地顯示其狀態。

　

集合屬性編輯器

IDE 默認的集合編輯器只允許用擴展方式編輯子項，該編輯器允許用戶使用下拉清單的方式進行編輯，同時還支援直接輸入集合值。

　

字串、標題屬性編輯器

該編輯器允許在 Caption 及其它 String 屬性中輸入多行文本。

參見 多行文本編輯器。

　

Hint 屬性編輯器

該編輯器允許在 Hint 屬性中輸入多行文本，支援長格式和短格式的 Hint。

參見 多行文本編輯器。

　

字串列表屬性編輯器

該編輯器擴展了 IDE 默認的字串列表屬性編輯器，支援查找、保存、裝載等操作。

參見 多行文本編輯器。

　

檔案名屬性編輯器

該編輯器允許用檔對話方塊來選擇 FileName 屬性的值。

　

Constraints屬性編輯器

該編輯器可以方便地修改控制項的 Constraints 屬性。

參見 Constraints 編輯器。

　

字體屬性編輯器

該編輯器可以在物件查看器中顯示字體的詳細資訊來取代默認的 (TFont) 顯示。

　

控制項捲軸屬性編輯器

該編輯器可以顯示捲軸的詳細資訊，包括是否可見，以及位置等。

　

Align 屬性編輯器

該編輯器可以為 Align 屬性顯示一個直觀的圖形。

　

元件名稱屬性編輯器

該編輯器調用 元件首碼專家 中的功能來為元件規範化重命名。
